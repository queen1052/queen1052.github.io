```

```

No sub-path. Vite `base` is `/`.

## Routes

| Route | Component | Description |
|---|---|---|
| `/` | `Home` | Post list with sidebar category/tag filters |
| `/post/:slug` | `PostDetail` | Full post content. `:slug` = filename without `.md` |

### Slug format

The `:slug` parameter is derived directly from the post's filename (without extension):

```
Filename:  site/posts/2026-04-09-modern-web-development.md
Slug:      2026-04-09-modern-web-development
Full URL:  https://queen1052.github.io/post/2026-04-09-modern-web-development
```

### Deep-link behaviour on GitHub Pages

GitHub Pages does not support server-side routing. Direct access to `/post/:slug` is handled
by the `404.html` SPA fallback mechanism:

1. Browser requests `https://queen1052.github.io/post/my-slug`
2. GitHub Pages finds no matching file → serves `404.html` (= copy of `index.html`)
3. React app boots in `404.html` context
4. `createBrowserRouter` reads `window.location.pathname` = `/post/my-slug`
5. Router matches `/post/:slug` route → renders `PostDetail` with `slug = "my-slug"`

No special redirect scripts or `window.history` hacks are required.

## Static Assets

Static files served from `dist/` root:

| Path | File | Source |
|---|---|---|
| `/robots.txt` | `dist/robots.txt` | `site/public/robots.txt` (copied by Vite) |
| `/sitemap.xml` | `dist/sitemap.xml` | Generated by Vite plugin `writeBundle` hook |
| `/feed.xml` | `dist/feed.xml` | Generated by Vite plugin `writeBundle` hook |
| `/.nojekyll` | `dist/.nojekyll` | `site/public/.nojekyll` (prevents Jekyll on gh-pages) |
| `/assets/*` | `dist/assets/*` | Vite-bundled CSS/JS with content-hash filenames |
| `/assets/img/*` | `dist/assets/img/*` | Post cover images from `site/public/assets/img/` |

## `sitemap.xml` Format

Generated at build time. Contains one `<url>` entry per published post plus the root URL:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://queen1052.github.io/</loc>
    <changefreq>daily</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://queen1052.github.io/post/2026-04-09-modern-web-development</loc>
    <lastmod>2026-04-09</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>
</urlset>
```

## `feed.xml` Format (RSS 2.0)

Generated at build time. Limited to the 20 most recent posts:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<rss version="2.0" xmlns:atom="http://www.w3.org/2005/Atom">
  <channel>
    <title>queen1052 Dev Blog</title>
    <link>https://queen1052.github.io</link>
    <description>개발 블로그</description>
    <atom:link href="https://queen1052.github.io/feed.xml" rel="self" type="application/rss+xml"/>
    <item>
      <title>모던 웹 개발의 미래</title>
      <link>https://queen1052.github.io/post/2026-04-09-modern-web-development</link>
      <guid>https://queen1052.github.io/post/2026-04-09-modern-web-development</guid>
      <pubDate>Wed, 09 Apr 2026 00:00:00 +0000</pubDate>
      <description><![CDATA[최신 웹 기술 트렌드와 함께...]]></description>
    </item>
  </channel>
</rss>
```
