## 소개

**GBrain**은 Vannevar Bush가 1945년 에세이 "As We May Think"에서 상상했던 memex를 AI 시대에 맞게 구현한 Postgres 기반 개인 지식 뇌(knowledge brain)입니다. 단순한 노트 앱이 아니라, AI 에이전트의 장기 기억 백본으로 설계된 시스템입니다.

GBrain의 핵심 철학은 **복리 효과**입니다. 일반적인 검색은 매번 처음부터 시작하지만, GBrain은 에이전트가 읽고 쓸수록 지식이 축적되고 연결되는 구조를 만듭니다.

---

## 주요 기능

GBrain이 제공하는 핵심 기능들:

- **하이브리드 검색**: 벡터 검색 + 키워드 검색 + RRF 퓨전으로 최적 결과 제공
- **다중 청킹 전략**: Recursive, Semantic(코사인 유사도), LLM 가이드 청킹 지원
- **그래프 순회**: 재귀 CTE를 활용한 링크 그래프 탐색 (기본 깊이 5)
- **임베딩 자동화**: OpenAI text-embedding-3-large로 자동 임베딩 생성
- **증분 동기화**: 변경된 파일만 처리하는 git-to-brain 동기화
- **원격 MCP 서버**: Claude Desktop, Perplexity 등 어떤 AI 클라이언트에서도 접근 가능
- **스킬 패키지**: ingest, query, maintain, enrich, briefing, migrate 등 제공

---

## 핵심 기능 1: 3단계 청킹 전략

GBrain의 가장 독특한 기술적 특징은 콘텐츠 유형에 따라 다른 청킹 전략을 사용한다는 점입니다:

**Recursive (타임라인, 대량 임포트)**
- 5단계 구분자 계층 (단락 → 줄 → 문장 → 절 → 단어)
- 300단어 청크, 50단어 문장 인식 오버랩
- 빠르고 예측 가능하며 손실 없음

**Semantic (컴파일된 진실)**
- 각 문장 임베딩 후 인접 코사인 유사도 계산
- Savitzky-Golay 평활화로 주제 경계 탐지
- 실패 시 Recursive로 폴백, 최고 품질

**LLM 가이드 (고가치 콘텐츠)**
- 128단어 후보로 사전 분할 → Claude Haiku에 주제 전환 파악 요청
- 슬라이딩 윈도우, 3회 재시도
- 가장 비용이 높지만 최상의 결과

## 핵심 기능 2: 프로덕션 에이전트 패턴 (GBRAIN_SKILLPACK)

GBrain의 진가는 단순 도구가 아닌 **에이전트 플레이북**에 있습니다. `GBRAIN_SKILLPACK.md`는 에이전트가 GBrain을 어떻게 사용해야 하는지 상세히 기술합니다:

**Brain-Agent 루프:**
- 응답 전 반드시 Brain에서 읽기
- 학습 후 반드시 Brain에 쓰기

**엔티티 감지 파이프라인:**
- 모든 메시지에서 사람/회사/아이디어 추출
- 7단계 Enrichment 프로토콜 (API 비용 티어별)

**20+ 자동화 크론 스케줄:**
- 오래된 페이지 감지 및 갱신
- 고아 페이지 정리
- 임베딩 커버리지 유지

이 패턴을 따르면, 7500개 페이지 뇌 기준으로 쿼리당 원본 파일 읽기 대비 ***71.5배 적은 토큰***을 사용합니다.

## 기술 스택 및 아키텍처

**데이터베이스:** PostgreSQL + pgvector (Supabase Pro 권장, $25/월)
**임베딩:** OpenAI text-embedding-3-large
**청킹 보조:** Anthropic Claude Haiku
**언어:** TypeScript 96.7%, PLpgSQL 2.7%

7,500 페이지 기준 스토리지 추정:
- 페이지 텍스트: ~150MB
- 임베딩 (22K x 1536 float): ~134MB
- HNSW 인덱스: ~270MB
- **총합 ~750MB** → Supabase Pro 필수

## 설치 및 사용법

```bash
# CLI 설치
bun add -g github:garrytan/gbrain

# 초기 설정 (Supabase 연동)
gbrain init --supabase

# 마크다운 임포트
gbrain import ~/git/brain/

# 검색
gbrain query "경쟁 역학에 대해 무엇을 알고 있나?"
```

**OpenClaw/Hermes Agent로 설치 (권장):**
```
Set up gbrain (https://github.com/garrytan/gbrain) as my knowledge brain.
1. Run: gbrain init --supabase
2. Import your documents
...
```

에이전트에 자연어로 설치를 요청하면 전체 설정을 자동으로 진행합니다.

## 마치며

GBrain은 AI 에이전트가 단기 기억의 한계를 넘어 장기적으로 성장하는 지식을 가질 수 있게 해주는 인프라입니다. 복리처럼 쌓이는 지식, 소스 추적 가능한 사실들, 그리고 Postgres 기반의 신뢰할 수 있는 저장소가 결합된 이 프로젝트는 AI 에이전트 생태계에서 가장 실용적인 장기 기억 솔루션 중 하나입니다.
