## 소개

**Memvid**는 AI 에이전트를 위한 초경량 단일 파일 메모리 레이어입니다. Rust로 98.5% 구현된 이 프로젝트는 GitHub에서 14,800개 이상의 스타를 획득했으며, Apache-2.0 라이선스로 배포됩니다.

기존의 복잡한 벡터 데이터베이스 설정 없이 단일 `.mv2` 파일 하나로 AI 에이전트의 장기 메모리를 관리할 수 있습니다. LoCoMo 롱컨텍스트 메모리 벤치마크에서 SOTA 대비 +35% 성능 향상을 달성했으며, P50 레이턴시 0.025ms라는 경이로운 속도를 자랑합니다.

## 주요 기능

- **단일 파일 포맷(.mv2)**: 복잡한 DB 설정 없이 하나의 파일로 모든 메모리 관리
- **Smart Frames**: 불변(immutable) 메모리 단위로 데이터 무결성 보장
- **+35% SOTA on LoCoMo**: 롱컨텍스트 메모리 벤치마크에서 현재 최고 성능
- **0.025ms P50 레이턴시**: Rust 네이티브 성능으로 실시간 에이전트 응답 지원
- **시간여행 디버깅**: 과거 특정 시점의 메모리 상태로 되돌아가 디버깅 가능
- **5개 SDK 지원**: CLI, Node.js, Python, Rust, Crates 생태계 완벽 지원

## Smart Frames 심층 분석

Smart Frames는 Memvid의 핵심 데이터 단위입니다. 각 프레임은 불변(immutable)으로 생성 후 수정되지 않으며, 이는 에이전트가 특정 시점의 메모리 상태를 정확히 재현할 수 있음을 의미합니다.

버전 관리 시스템(Git)처럼 모든 메모리 변경사항이 추적되고, 과거 어느 시점으로도 롤백이 가능합니다. 이 특성은 디버깅뿐만 아니라 에이전트 행동 감사(audit)에도 중요한 역할을 합니다.

임베딩 벡터와 메타데이터가 함께 저장되어, 시맨틱 검색과 정확한 키-값 검색을 모두 지원합니다. Rust의 메모리 안전성 덕분에 대용량 메모리 조작 시에도 안정적으로 동작합니다.

## 성능 벤치마크

Memvid는 LoCoMo(Long-Context Memory Operations) 벤치마크에서 기존 접근법 대비 35% 향상된 성능을 달성했습니다. 이는 롱컨텍스트 AI 응용에서 메모리 검색 정확도가 크게 개선되었음을 의미합니다.

속도 측면에서는 P50 레이턴시 0.025ms(25마이크로초)를 기록합니다. 이는 초당 40,000회 이상의 메모리 조회가 가능한 수준으로, 고빈도 에이전트 루프에서도 메모리가 병목이 되지 않습니다.

## 기술 스택 및 아키텍처

- **언어**: Rust 98.5% (핵심 엔진), 기타 2%
- **라이선스**: Apache-2.0
- **파일 포맷**: .mv2 (Memvid v2)
- **인덱싱**: HNSW(Hierarchical Navigable Small Worlds) 기반 근사 최근접 이웃 검색
- **임베딩**: 플러그인 방식으로 다양한 임베딩 모델 지원
- **SDK**: CLI, Node.js(npm), Python(pip), Rust(cargo), Crates

## 설치 및 사용법

Rust 프로젝트에 추가:
```toml
# Cargo.toml
[dependencies]
memvid-core = "0.2"
```

```bash
cargo add memvid-core
```

Python SDK 사용:
```python
from memvid import Memvid

# 메모리 생성
mv = Memvid("agent_memory.mv2")

# 메모리 쓰기
mv.remember("사용자 이름은 김철수이다", metadata={"category": "user_info"})

# 시맨틱 검색
results = mv.search("사용자 이름", top_k=3)

# 특정 시점 메모리 조회 (시간여행)
past_state = mv.at_timestamp("2026-04-01T12:00:00")
```

CLI 사용:
```bash
memvid search "찾고 싶은 내용" --file agent_memory.mv2
memvid stats agent_memory.mv2
```

## 마치며

**Memvid**는 복잡한 벡터 DB 설정 없이 AI 에이전트에 강력한 장기 메모리를 부여할 수 있는 가장 빠르고 간단한 솔루션입니다. LoCoMo 벤치마크 SOTA와 0.025ms 레이턴시는 단순한 마케팅 수치가 아닌 Rust 네이티브 구현의 실제 성과입니다. 단일 파일 포맷과 5개 SDK 지원으로 모든 환경의 AI 에이전트에 즉시 통합할 수 있습니다.
