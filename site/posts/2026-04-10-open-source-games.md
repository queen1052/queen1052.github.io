## 소개

leereilly/games는 Lee Reilly가 관리하는 **오픈소스 HTML5 브라우저 게임 큐레이션 저장소**입니다. 아케이드, 1인칭 슈팅(FPS), RPG, 퍼즐, 전략, 스포츠 등 다양한 장르의 브라우저 게임들을 체계적으로 분류하고 링크를 제공합니다. 설치 없이 브라우저에서 바로 플레이할 수 있는 모든 게임은 오픈소스이며, 대부분 수정과 재배포가 자유롭습니다.

이 저장소의 가치는 단순한 게임 목록이 아닙니다. **게임 개발 학습 자료**로서 활용도가 높습니다. 각 게임의 GitHub 소스 코드를 직접 탐색하면서 어떻게 2D 렌더링, 물리 엔진, 게임 루프, 충돌 감지, 네트워킹이 구현되었는지 학습할 수 있습니다. 특히 JavaScript/TypeScript로 게임 개발을 시작하는 학습자에게 풍부한 실전 코드 예시를 제공합니다.

또한 게임 엔진과 프레임워크(Phaser, Three.js, Babylon.js, PlayCanvas 등) 섹션도 포함되어 있어 자체 게임을 만들 때 어느 엔진을 선택해야 하는지 한눈에 비교할 수 있습니다.

## 주요 기능

- **장르별 분류**: 아케이드, 육상 게임, FPS, RPG/어드벤처, 전략, 퍼즐, 스포츠, 레이싱 등 장르별로 게임을 분류합니다. 원하는 장르를 쉽게 탐색합니다.
- **브라우저 직접 실행**: 모든 게임이 HTML5 기반으로 설치 없이 브라우저에서 바로 플레이 가능합니다. 데스크탑과 모바일 모두 지원하는 게임도 다수 포함됩니다.
- **오픈소스 소스 코드**: 모든 게임의 GitHub 저장소 링크가 제공됩니다. 코드를 읽고, 포크하고, 수정해 자신만의 버전을 만들 수 있습니다.
- **게임 엔진/프레임워크 목록**: Phaser(2D 게임 엔진), Three.js(3D WebGL), Babylon.js(3D), PixiJS(2D 렌더러), PlayCanvas 등 인기 엔진/라이브러리를 정리합니다.
- **멀티플레이어 게임**: Socket.io나 WebRTC를 사용한 실시간 멀티플레이어 게임도 포함됩니다. 네트워크 게임 구현을 학습하는 좋은 예시들입니다.
- **지속적 업데이트**: 커뮤니티 기여를 통해 지속적으로 새 게임이 추가됩니다.

### 인기 게임 하이라이트

**아케이드:**
- **3D Tetris** — Babylon.js로 구현한 3D 테트리스
- **Pac-Man HTML5** — 클래식 팩맨의 완벽한 HTML5 재현
- **Angry Birds 클론** — Box2D 물리 엔진 기반 앵그리버드

**전략:**
- **0 A.D.** — Civilization 스타일의 역사 전략 게임
- **FreeCiv** — 웹 버전 문명 게임
- **Tanks** — 멀티플레이어 탱크 전투

**퍼즐/캐주얼:**
- **2048** — 유명한 숫자 퍼즐 게임의 오픈소스 원본
- **Hexgl** — WebGL 고속 레이싱 게임 (미래지향적 그래픽)
- **CrossCode** — 풀 RPG (Phaser 기반)

### 게임 엔진 비교 가이드

| 엔진 | 특징 | 적합한 게임 |
|------|------|------------|
| Phaser | 2D 특화, 학습 자료 풍부 | 플랫포머, 아케이드 |
| Three.js | 3D WebGL, 낮은 수준 | 3D 비주얼, 커스텀 렌더링 |

| PixiJS | 2D 렌더러, 최고 성능 | 2D 고성능 렌더링 |
| PlayCanvas | 클라우드 에디터 통합 | 팀 협업 게임 개발 |

## 사용 방법

```bash
# 저장소 탐색
git clone https://github.com/leereilly/games
cd games
# README.md에서 게임 목록 탐색

# 특정 게임 클론 및 실행 예시 (Phaser 게임)
git clone https://github.com/example/phaser-platformer
cd phaser-platformer
# index.html을 브라우저에서 열거나
npx http-server . -p 8080
# http://localhost:8080 에서 플레이

# PR로 새 게임 기여
# 1. 포크 후 README.md에 게임 링크 추가
# 2. 카테고리 맞는 섹션에 삽입
# 3. 형식 맞춰 PR 제출
```

## 활용 사례 / 사용 시나리오

1. **게임 개발 학습**: 좋아하는 장르의 오픈소스 게임 코드를 탐색합니다. "이 플랫포머 게임은 점프 물리를 어떻게 구현했을까?" 같은 질문을 실제 코드로 답합니다. 특히 Phaser 기반 게임들이 학습에 좋습니다.

2. **게임 잼(Game Jam) 준비**: 게임 잼 전에 유사한 장르의 오픈소스 게임을 탐색해 기술 스택과 구현 패턴을 파악합니다. 처음부터 시작하는 대신 검증된 구조를 참조합니다.

3. **교육용 게임 콘텐츠**: 오픈소스 게임을 학교나 교육 플랫폼에 허가 문제 없이 통합합니다. 코딩 교육에서 학생들이 직접 게임 코드를 수정하는 프로젝트로 활용합니다.

## 결론

leereilly/games는 HTML5 오픈소스 게임 세계의 가장 포괄적인 큐레이션입니다. 단순히 게임을 즐기는 것을 넘어 게임 개발 학습 자료로서의 가치가 탁월합니다. 수백 가지 게임의 소스 코드를 탐색하면서 다양한 게임 기법과 엔진 사용법을 자연스럽게 익힐 수 있습니다. 브라우저 게임 개발에 관심 있는 모든 개발자에게 필수 북마크입니다.
