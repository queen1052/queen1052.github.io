## 소개

[n8n](https://github.com/n8n-io/n8n)("nodemation"의 약어, "n-eight-n"으로 발음)은 기술 팀을 위한 AI 네이티브 워크플로우 자동화 플랫폼입니다. 코드의 유연성과 노코드의 속도를 결합해, 복잡한 비즈니스 프로세스 자동화를 시각적 인터페이스와 JavaScript/Python 코드 모두로 구현할 수 있습니다.

GitHub에서 184k 스타를 기록한 n8n은 공정 코드(Fair-code) 라이선스 하에 배포되어 소스 코드는 항상 공개되며 어디서나 자체 호스팅이 가능합니다. 400개 이상의 통합, 900개 이상의 즉시 사용 가능한 워크플로우 템플릿, LangChain 기반의 완전한 AI 에이전트 빌딩 기능을 제공합니다.

## 주요 기능

n8n은 자동화 플랫폼의 넓이와 깊이를 모두 갖추고 있습니다:

- **AI 네이티브**: LangChain 기반 AI 에이전트 워크플로우. 자신의 데이터와 모델로 구축
- **400개+ 통합**: Slack, Google Sheets, HubSpot, Stripe, Airtable 등 거의 모든 서비스 연결
- **코드 + 노코드**: 시각적 노드 편집기와 JavaScript/Python 코드 혼합 사용
- **완전한 제어**: Self-host로 데이터 완전 소유, Fair-code 라이선스
- **엔터프라이즈**: 고급 권한, SSO, 에어갭 배포
- **9백 개 템플릿**: 즉시 사용 가능한 워크플로우 컬렉션
- **비주얼 캔버스**: 직관적인 ReactFlow 기반 워크플로우 에디터

## 핵심 기능 심층 분석: AI 에이전트 워크플로우

n8n의 AI 에이전트 빌딩은 LangChain의 강력함을 시각적 인터페이스로 접근 가능하게 만듭니다. 복잡한 코딩 없이 멀티 스텝 AI 에이전트를 노드로 연결해 구성할 수 있습니다.

**전형적인 AI 에이전트 워크플로우:**
1. **트리거**: 이메일 수신, API 호출, 스케줄 등
2. **데이터 수집**: 관련 정보를 여러 소스에서 수집
3. **AI 처리**: LLM으로 분석, 요약, 의사결정
4. **도구 실행**: 웹 검색, 데이터베이스 쿼리, API 호출
5. **결과 배포**: Slack 알림, 이메일 발송, 데이터베이스 업데이트

예를 들어 "영업 리드 자동 분류 에이전트"를 만들면: 신규 리드 이메일 수신 → 웹 검색으로 기업 정보 수집 → GPT-4로 우선순위 분류 → CRM 업데이트 → 영업팀 Slack 알림이 자동으로 진행됩니다.

## 핵심 기능 심층 분석: 코드 노드와 유연성

n8n의 핵심 차별점 중 하나는 "코드가 필요할 때 코드를 쓸 수 있다"는 것입니다:

**Function 노드:**
```javascript
// 모든 Node.js 코드 실행 가능
const items = $input.all();
return items.map(item => {
  return {
    json: {
      processedValue: item.json.value * 2,
      timestamp: new Date().toISOString()
    }
  };
});
```

npm 패키지 가져오기도 지원합니다. 비주얼 노드로 해결되지 않는 복잡한 데이터 변환, 커스텀 로직, 서드파티 라이브러리 활용이 모두 가능합니다. Python 코드도 지원합니다.

AI 코드 빌더 기능(`ai-builder`)을 사용하면 자연어 설명으로 워크플로우를 자동 생성할 수도 있습니다.

## 기술 스택 및 아키텍처

- **언어**: TypeScript (91.4%) + Vue.js (7.3%)
- **런타임**: Node.js 24+
- **UI**: Vue 3 + Vite (시각적 워크플로우 에디터)
- **빌드**: Turborepo 모노레포
- **라이선스**: Sustainable Use License (Fair-code)

컨테이너: Docker 이미지 제공 (`docker.n8n.io/n8nio/n8n`)

## 설치 및 사용법

**npx로 즉시 시작 (Node.js 필요):**
```bash
npx n8n
```

**Docker:**
```bash
docker volume create n8n_data
docker run -it --rm --name n8n \
  -p 5678:5678 \
  -v n8n_data:/home/node/.n8n \
  docker.n8n.io/n8nio/n8n
```

에디터 접근: http://localhost:5678

**클라우드 버전:** app.n8n.cloud 에서 계정 생성 후 즉시 사용 가능.

## 마치며

n8n은 184k 스타와 646명의 컨트리뷰터가 만들어낸 워크플로우 자동화의 새로운 패러다임입니다. Zapier나 Make(Integromat)와 같은 SaaS 도구의 편의성과 완전한 데이터 제어 및 코드 유연성을 동시에 제공합니다. AI 에이전트 빌딩이 n8n의 핵심 기능으로 자리잡으면서, 단순 자동화를 넘어 지능형 비즈니스 프로세스 자동화 플랫폼으로 진화하고 있습니다. 기술 팀의 자동화 허브로 n8n을 선택하면 거의 모든 반복 작업을 효율적으로 자동화할 수 있습니다.
