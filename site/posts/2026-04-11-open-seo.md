## 소개: 당신이 완전히 통제하는 SEO 도구

"SEO 툴 for the people." OpenSEO의 슬로건이다.

**OpenSEO**는 every-app(Ben Senescu)이 개발한 오픈소스 SEO 도구다. GitHub **1.3k stars**, TypeScript 97.9%, MIT 라이선스로 공개되어 있다. Semrush나 Ahrefs 같은 고가의 SEO 서비스에 대한 대안으로, 사용자가 직접 호스팅하고 DataForSEO API의 종량제 비용만 지불하는 방식이다.

앱 자체는 완전 무료다($0). DataForSEO API 사용 비용만 발생하며, 신규 가입 시 $1 무료 크레딧이 제공되어 바로 테스트해볼 수 있다. 최소 충전 금액은 $50이다.

## 주요 기능: 핵심 SEO 워크플로우

OpenSEO는 복잡한 올인원 수트 대신, 가장 중요한 4개의 SEO 워크플로우에 집중한다.

**키워드 리서치**: 공략할 가치 있는 토픽을 찾고, 검색 수요를 추정하고, 다음에 작성할 콘텐츠를 우선순위화한다. DataForSEO Labs API를 사용하며, 기본값 150개 결과 기준 쿼리당 약 $0.035가 소요된다.

**도메인 인사이트**: 도메인이 검색 가시성을 얻거나 잃는 부분을 파악해 매출에 영향을 미치는 페이지에 집중할 수 있게 한다. 200개 랭킹 키워드 기준 도메인당 약 $0.04가 소요된다.

**백링크 분석**: 사이트에 링크를 다는 곳, 링크를 많이 받는 페이지, 새로 획득하거나 잃은 링크를 파악한다. 도메인 기준 약 $0.06, 페이지 기준 약 $0.04가 소요된다.

**사이트 감사**: 기술적 문제를 조기 발견해 크롤링과 랭킹이 쉽도록 한다. 현재 구현에서는 유료 API 호출 없이 작동한다.

**로드맵**: 랭크 트래킹과 AI SEO(LLM Citations) 기능이 예정되어 있다.

## 비용 구조 심층 분석

OpenSEO의 가장 큰 장점은 완전한 비용 투명성이다.

**사이트 감사**: 무료 (유료 API 호출 없음)

**키워드 리서치**:
- 150개 결과: $0.035/쿼리
- 300개 결과: $0.05/쿼리
- 500개 결과: $0.07/쿼리

**도메인 오버뷰**: 약 $0.04/도메인 (상위 200개 랭킹 키워드 포함)

**백링크 검색**:
- 도메인 기본 검색: ~$0.06
- 페이지 기본 검색: ~$0.04
- 추가 탭(Referring Domains, Top Pages) 각각: +$0.02

**계획 예시**: 100개 도메인 오버뷰 = $4.01. 100개 키워드 리서치(150 결과) = $3.50. 이는 Semrush의 월 $119.95 또는 Ahrefs의 월 $129에 비해 훨씬 비용 효율적이다.

## 셀프호스팅 방법

**Docker 셀프호스팅** (권장):

```bash
# 1. 환경 파일 설정
cp .env.example .env

# 2. DataForSEO API 키 설정
# DataForSEO 로그인과 API 비밀번호를 base64로 인코딩
printf '%s' 'YOUR_LOGIN:YOUR_PASSWORD' | base64
# 결과를 .env의 DATAFORSEO_API_KEY에 입력

# 3. 실행
docker compose up -d

# 4. 접근
# http://localhost:3001 (기본값)

# 최신 이미지로 업데이트
docker compose pull && docker compose up -d
```

**Cloudflare 셀프호스팅**: 여러 기기나 팀에서 사용할 경우 적합하다. Cloudflare 계정에 "Deploy to Cloudflare" 버튼으로 배포하며, 프리 플랜에서도 작동한다. Cloudflare D1(데이터베이스), R2(캐시 스토리지), KV(데이터)를 사용해 안정적인 SaaS 같은 경험을 제공한다.

두 방식의 선택 기준: Docker는 단일 사용자의 홈랩이나 로컬 사용에 적합하고, Cloudflare는 여러 기기나 팀에서 접근할 때 적합하다.

## 기술 스택 및 로컬 개발

**기술 스택**: TypeScript 97.9%로 풀스택 TypeScript 프로젝트다. Cloudflare Workers(엣지 서빙), Drizzle ORM(데이터베이스), Vite(빌드), pnpm(패키지 관리), Vitest(테스트), Better Auth(인증—호스팅 모드).

**인증 모드**:
- `AUTH_MODE=cloudflare_access`: Cloudflare Access JWT 검증 (기본값)
- `AUTH_MODE=local_noauth`: 로컬 개발용 인증 없음, `admin@localhost` 자동 주입
- `AUTH_MODE=hosted`: Better Auth 이메일/비밀번호 인증 (hosted 모드)

**로컬 개발**:

```bash
# 사전 요구사항: Node.js 20+, pnpm, DataForSEO API 키
pnpm install
pnpm run db:migrate:local  # 최초 1회만

# 환경 설정
cp .env.example .env.local
# DATAFORSEO_API_KEY 설정 (base64 인코딩된 login:password)

# 개발 서버 실행
pnpm run dev

# 에이전트 디버깅 편의 모드 (권장)
mkdir .logs
touch .logs/dev-server.log
pnpm dev:agents  # portless로 실행, http://open-seo.localhost:1355
```

AI 네이티브 설계: Claude Code나 Cowork 같은 AI 도구와 함께 사용하면 다른 플랫폼보다 강력한 AI SEO 기능을 올릴 수 있도록 아키텍처가 설계되어 있다.

## DataForSEO API 키 설정

OpenSEO는 DataForSEO를 통해 SEO 데이터를 가져온다. 설정 방법:

1. `app.dataforseo.com/api-access`에서 API 자격증명 요청
2. 이메일로 받은 로그인 + API 비밀번호로 base64 인코딩:
   ```bash
   printf '%s' 'YOUR_LOGIN:YOUR_API_PASSWORD' | base64
   ```
3. 결과값을 `DATAFORSEO_API_KEY` 환경 변수에 설정
4. 백링크 기능은 DataForSEO 백링크 구독(트라이얼 또는 유료)도 필요

신규 계정에는 $1 무료 크레딧이 제공되어 기본 기능을 즉시 테스트할 수 있다.

## 마치며

OpenSEO는 "SEO 도구는 비싸야 한다"는 고정관념을 깨는 프로젝트다. 오픈소스로 앱 자체는 무료이고, DataForSEO의 종량제 API를 통해 필요한 만큼만 비용을 지불한다. Docker로 홈랩에 쉽게 설치하거나, Cloudflare의 프리 플랜으로 팀 전체가 사용할 수 있다.

복잡한 기능 세트 대신 키워드 리서치, 도메인 인사이트, 백링크 분석, 사이트 감사 네 가지 핵심 워크플로우에 집중하는 것이 OpenSEO의 강점이다. SEO를 시작하거나, 기존 서비스의 구독 비용을 줄이고 싶다면 OpenSEO가 좋은 선택이 될 것이다.
