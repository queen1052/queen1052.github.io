## 소개

**OWL(Optimized Workforce Learning)**은 실세계 작업 자동화를 위한 멀티에이전트 협업 프레임워크입니다. GitHub에서 19,600개의 스타와 2,300개의 포크를 기록했으며, Python 94.1%로 구현된 Apache-2.0 라이선스 프로젝트입니다.

GAIA(General AI Assistants) 벤치마크에서 69.09점으로 오픈소스 프레임워크 중 1위를 달성했으며, NeurIPS 2025에 채택된 학술적으로도 인정받은 프로젝트입니다. CAMEL-AI 프레임워크 위에 구축되어, 동적 에이전트 상호작용으로 자연스럽고 효율적인 작업 자동화를 구현합니다.

## 주요 기능

- **GAIA 벤치마크 오픈소스 1위**: 69.09점으로 실세계 작업 처리 능력 최상위 검증
- **온라인 검색**: Wikipedia, Google, DuckDuckGo, Baidu, Bocha 등 실시간 정보 수집
- **브라우저 자동화**: Playwright 기반 스크롤·클릭·입력·다운로드·내비게이션
- **멀티모달 처리**: 인터넷/로컬 동영상·이미지·오디오 처리
- **문서 파싱**: Word, Excel, PDF, PowerPoint → 텍스트/마크다운 변환
- **코드 실행**: Python 인터프리터로 동적 코드 작성 및 실행
- **MCP 통합**: MCP 프로토콜로 외부 도구·데이터소스와 표준화된 연동

## GAIA 벤치마크란?

GAIA(General AI Assistants)는 AI 에이전트의 실세계 문제 해결 능력을 평가하는 벤치마크입니다. 단순한 QA가 아닌, 웹 검색, 문서 분석, 코드 실행, 도구 사용을 복합적으로 요구하는 현실적 과제들로 구성됩니다.

OWL의 69.09점은 상용 AI 에이전트들과 경쟁하는 수준이며, 오픈소스 프레임워크 중 단연 1위입니다. 이 성과는 단순한 LLM 호출이 아닌, 체계적인 멀티에이전트 협업 아키텍처의 우수성을 증명합니다.

## 멀티에이전트 협업 아키텍처

OWL의 핵심은 역할 기반 에이전트 협업입니다. 각 에이전트는 특정 역할(사용자·보조자·도구 전문가)을 담당하며, 작업에 따라 동적으로 팀을 구성합니다.

**사용 가능한 주요 툴킷:**
- BrowserToolkit: Playwright 기반 웹 브라우저 자동화
- VideoAnalysisToolkit: 동영상 내용 분석
- AudioAnalysisToolkit: 음성 분석 (OpenAI API 필요)
- CodeExecutionToolkit: Python 코드 실행
- SearchToolkit: DuckDuckGo, Google, Wikipedia 검색
- DocumentProcessingToolkit: PDF, DOCX 문서 파싱
- ArxivToolkit, GitHubToolkit, GoogleMapsToolkit 등 전문 툴킷

## 기술 스택 및 아키텍처

- **언어**: Python 94.1%, Shell 2.8%, Dockerfile 0.6%
- **라이선스**: Apache-2.0
- **기반 프레임워크**: CAMEL-AI
- **브라우저**: Playwright (Chrome, Edge, Chromium 지원)
- **웹 UI**: Gradio (한국어·영어·일본어 버전)
- **MCP**: Node.js 기반 MCP Toolkit 지원
- **컨테이너**: Docker + docker-compose

## 설치 및 사용법

```bash
# uv 사용 (권장)
git clone https://github.com/camel-ai/owl.git
cd owl
pip install uv
uv venv .venv --python=3.10
source .venv/bin/activate   # Windows: .venv\Scripts\activate
uv pip install -e .
```

환경 변수 설정:
```bash
cp .env_template .env
# .env 파일에서 API 키 설정
# OPENAI_API_KEY, ANTHROPIC_API_KEY 등
```

실행:
```bash
# 기본 실행 (OpenAI)
python examples/run.py

# Claude로 실행
python examples/run_claude.py

# 웹 UI 실행 (영어)
python owl/webapp.py
```

Docker로 실행:
```bash
docker compose up -d
docker compose exec owl bash
source .venv/bin/activate
python examples/run.py
```
