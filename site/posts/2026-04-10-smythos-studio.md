## 소개

SmythOS Studio는 SmythOS 팀이 공개한 **AI 에이전트 비주얼 빌더**입니다. TypeScript 41.7%, CSS 35.2%, JavaScript 22.2%로 구성된 풀스택 웹 애플리케이션으로, 노-코드/로-코드 접근 방식으로 AI 에이전트를 설계·배포할 수 있습니다. `pnpm dev` 한 줄로 로컬 개발 서버를 시작할 수 있어 설정이 간단합니다.

SmythOS Studio의 핵심 가치는 **시각적 설계와 실제 배포 사이의 간극을 줄이는 것**입니다. 기존 에이전트 프레임워크들은 코드 작성에 집중하지만, SmythOS는 그래픽 인터페이스에서 에이전트 플로우, 도구 연결, 조건 논리를 설계하고 즉시 배포할 수 있게 합니다. 비개발자도 AI 에이전트를 만들 수 있고, 개발자는 빠른 프로토타이핑 도구로 활용할 수 있습니다.

MySQL 데이터베이스 백엔드와 RESTful API를 통해 기업 환경의 데이터 소스와 연동하고, 에이전트 실행 이력과 상태를 영구적으로 저장할 수 있습니다.

## 주요 기능

- **드래그 앤 드롭 에이전트 빌더**: 노드 기반 시각적 편집기에서 에이전트 컴포넌트를 연결합니다. LLM 노드, 도구 노드, 조건 분기, 루프 등을 드래그로 배치하고 선으로 연결합니다.
- **멀티 LLM 지원**: OpenAI GPT, Anthropic Claude, Google Gemini 등 다양한 LLM을 노드로 연결할 수 있습니다. 최적 LLM을 태스크별로 선택하거나 폴백 체인을 구성합니다.
- **도구 마켓플레이스**: 사전 구축된 도구 라이브러리(검색, 코드 실행, 파일 처리, API 호출 등)를 드래그로 에이전트에 추가합니다.
- **실시간 에이전트 테스트**: 빌더 내에서 에이전트를 즉시 테스트하고 각 노드의 입출력을 실시간으로 시각화합니다.
- **MySQL 기반 영구 저장**: 에이전트 정의, 실행 이력, 사용자 데이터를 MySQL에 저장합니다. 팀 단위로 에이전트를 공유하고 버전 관리합니다.
- **배포 자동화**: 완성된 에이전트를 API 엔드포인트, SlackBot, 웹사이트 위젯 등 다양한 형태로 즉시 배포합니다.

### 비주얼 빌더 — 에이전트 설계의 민주화

SmythOS Studio의 비주얼 빌더는 React Flow 기반으로 구현됩니다. 좌측 패널에서 사용 가능한 노드 유형을 탐색하고, 캔버스에 드래그해 에이전트 플로우를 설계합니다. 각 노드는 설정 패널에서 LLM 모델, 시스템 프롬프트, 도구 옵션을 구성할 수 있습니다.

조건 분기 노드는 에이전트가 상황에 따라 다른 경로를 선택하도록 합니다. 예를 들어 "사용자 질문이 기술 관련인가?"를 판단해 기술 전문 에이전트로 라우팅하거나 일반 에이전트로 처리하는 로직을 코드 없이 구성할 수 있습니다.

### MySQL 연동 — 기업 데이터와의 통합

```yaml
# database.config.yaml 설정 예시
database:
  host: localhost
  port: 3306
  name: smythos_studio
  user: ${DB_USER}
  password: ${DB_PASSWORD}
```

에이전트 실행 이력, 대화 기록, 사용자 피드백이 MySQL에 저장됩니다. 기업 BI 도구로 에이전트 성능 분석도 가능합니다.

## 설치 / 사용법

```bash
# 저장소 클론
git clone https://github.com/SmythOS/smythos-studio
cd smythos-studio

# 의존성 설치 (pnpm 필수)
pnpm install

# 환경 변수 설정
cp .env.example .env
# DATABASE_URL, OPENAI_API_KEY 등 설정

# 개발 서버 시작
pnpm dev

# 프로덕션 빌드
pnpm build
pnpm start
```

브라우저에서 `http://localhost:3000` 접속 후 비주얼 빌더를 사용할 수 있습니다.

## 활용 사례 / 사용 시나리오

1. **비개발자 에이전트 개발**: 제품 매니저, 마케터, 분석가가 코드 없이 자신의 업무에 맞는 AI 에이전트를 직접 만듭니다. 데이터 리포트 자동화, 고객 질문 분류기 등을 드래그 앤 드롭으로 구축합니다.

2. **빠른 에이전트 프로토타이핑**: 개발자가 새 에이전트 아이디어를 코드 없이 빠르게 프로토타이핑합니다. 비주얼 빌더에서 컨셉을 검증한 후 코드로 변환하거나 그대로 배포합니다.

3. **팀 에이전트 라이브러리 구축**: MySQL 저장 덕분에 팀 내에서 에이전트를 공유하고 재사용합니다. "고객 지원 에이전트", "데이터 분석 에이전트" 같은 팀 공용 에이전트를 한 곳에서 관리합니다.

## 결론

SmythOS Studio는 AI 에이전트 개발의 진입 장벽을 낮추는 비주얼 빌더입니다. TypeScript 기반의 안정적인 구현과 MySQL 통합이 기업 환경에서의 활용을 뒷받침합니다. 비개발자도 강력한 에이전트를 만들 수 있다는 점이 가장 큰 강점이며, 빠른 프로토타이핑 도구로도 개발자에게 유용합니다.
