## 소개

DeepTutor는 HKUDS 연구팀이 개발한 **에이전트 기반 개인화 학습 플랫폼**입니다. 단순한 Q&A 챗봇을 넘어, 학습자의 개인 자료와 지식 베이스를 중심으로 맞춤형 학습 경험을 제공하는 완전한 튜터링 시스템입니다. Python 3.11+ 백엔드(FastAPI)와 Next.js 16 프론트엔드로 구성되어 있으며, 복잡한 주제를 단계별 시각적 여정으로 분해하거나 AI와 공동 저술 작업을 할 수 있는 Co-Writer를 포함합니다.

기존 AI 튜터 도구들이 단발적인 답변에 그쳤다면, DeepTutor는 **지속적인 학습 생태계**를 구축합니다. 사용자의 노트, 문서, 대화 히스토리가 유기적으로 연결되어 학습이 진행될수록 시스템이 점점 더 개인화됩니다. 세션을 넘나들며 학습 여정을 이어갈 수 있다는 점이 다른 도구와 결정적으로 다릅니다.

최근 공개된 v1.0.0-beta.4에서는 다중 사용자 인증, 다양한 UI 테마, LightRAG 통합 등이 로드맵에 추가되었으며, 오픈소스 커뮤니티를 통해 빠르게 발전하고 있습니다.

## 주요 기능

- **Chat — 통합 지능형 워크스페이스**: 일반 질문부터 Deep Research까지 하나의 대화창에서 처리. 어느 시점이든 Deep Solve, Math Animator 등 다른 워크플로우로 에스컬레이션 가능합니다.
- **Deep Research**: 주제를 서브토픽으로 분해하고 RAG, 웹, 학술 논문에 병렬 연구 에이전트를 파견해 완전히 인용된 보고서를 생성합니다.
- **Math Animator**: 수학 개념을 Manim 기반 시각적 애니메이션과 스토리보드로 변환합니다. 미적분, 선형대수, 통계를 시각화하는 데 탁월합니다.
- **Co-Writer**: AI가 내장된 마크다운 에디터. 텍스트를 선택 후 Rewrite/Expand/Shorten을 실행할 수 있으며, 지식 베이스 또는 웹 컨텍스트를 선택해 활용할 수 있습니다. 전체 undo/redo를 지원하는 비파괴 편집 방식으로 동작합니다.
- **Guided Learning**: 개인 자료를 시각적·단계별 학습 여정으로 변환합니다. 각 단계마다 HTML 페이지(다이어그램, 예제 포함)를 자동 생성합니다.
- **TutorBot CLI**: 브라우저 없이 터미널에서 튜터 기능을 활용할 수 있는 REPL 인터페이스.

### Guided Learning — 개인 자료를 학습 로드맵으로

Guided Learning은 사용자가 주제를 입력하면 DeepTutor가 자동으로 3~5개의 단계별 지식 포인트를 설계합니다. 각 포인트는 시각적 HTML 페이지(다이어그램, 예제, 설명 포함)로 변환되며, 학습 중 컨텍스트 Q&A로 심층 탐구를 할 수 있습니다. 완료 시 학습 요약이 제공되고, 세션이 영구 저장되어 언제든 중단 후 재개할 수 있습니다.

노트북 레코드와 연동하면 자신이 이전에 학습한 내용(코드, 문서, 대화)을 베이스로 삼아 다음 단계를 설계할 수 있어, 누적 학습 효과가 탁월합니다.

### Deep Research — 병렬 에이전트 기반 리서치

Deep Research 모드는 입력된 주제를 여러 서브토픽으로 분해하고, 각각의 에이전트를 RAG(개인 지식 베이스), 웹, arXiv·Semantic Scholar 등 학술 논문 데이터베이스에 병렬로 파견합니다. 수집된 정보는 인용 검증 단계를 거쳐 할루시네이션 없이 완전히 인용된 보고서로 통합됩니다. 기존 에이전트 리서치 도구와 달리 개인 지식 베이스를 컨텍스트로 활용하기 때문에 사용자 특정 도메인에 맞춤화된 결과가 나옵니다.

## 기술 스택 / 아키텍처

| 구성요소 | 기술 |
|---------|------|
| Backend | Python 3.11+, FastAPI |
| Frontend | Next.js 16, React |
| 지식 베이스 | RAG (LightRAG 통합 예정) |
| 시각화 | Manim (수식 애니메이션) |
| 데이터베이스 | (설정 기반) |
| 배포 | Docker, 로컬 개발 |

아키텍처 흐름:  
`사용자 입력 → 워크플로우 선택 → 에이전트 파견 (RAG/웹/학술) → 결과 통합 → Co-Writer/Guided Learning/보고서 출력`

## 설치 / 사용법

```bash
# Python 환경 생성 (conda)
conda create -n deeptutor python=3.11
conda activate deeptutor

# 저장소 클론
git clone https://github.com/HKUDS/DeepTutor
cd DeepTutor

# 백엔드 실행 (FastAPI)
bash docker/launch_backend_service.sh

# 프론트엔드 설치 및 실행 (별도 터미널)
cd web
npm install
npm run dev
```

설정 시 `config.yaml`에 LLM 키(OpenAI 등), 임베딩 모델 설정이 필요합니다. Zep Cloud API 키를 `ZEP_API_KEY`로 설정하면 세션 간 장기 기억 기능이 활성화됩니다.

## 활용 사례 / 사용 시나리오

1. **대학원생/연구자**: 수십 편의 논문을 지식 베이스에 축적한 뒤, Deep Research로 특정 주제의 동향 보고서를 자동 작성합니다. 인용 검증 덕분에 할루시네이션 없는 신뢰할 수 있는 자료를 얻고, Co-Writer로 초안을 다듬습니다.

2. **개발자/기술 학습자**: 복잡한 코드베이스나 기술 문서를 튜터에게 학습시킨 뒤, Math Animator로 알고리즘을 시각화하고 TutorBot CLI를 통해 CI 파이프라인에서 자동 문서 질의를 실행합니다.

3. **자기 주도 학습자**: 유튜브 강의 노트, PDF 교재를 업로드하고 Guided Learning으로 시각적 학습 플랜을 받습니다. 세션이 영구 저장되어 한 달에 걸친 장기 학습 프로젝트도 일관성 있게 관리할 수 있습니다.

## 결론

DeepTutor는 단순한 AI 챗봇을 넘어 **개인화된 학습 인프라**를 구축하는 도구입니다. 개인 자료 기반 RAG와 영구 노트 연동이 핵심 강점으로, 장기 학습 프로젝트를 진행하는 학생과 연구자에게 적합합니다. 현재 v1.0.0 베타 단계라 다중 사용자 환경에 한계가 있지만, 인증 시스템과 LightRAG 통합이 로드맵에 있어 팀 활용도 조만간 가능해질 전망입니다. GitHub에서 별(⭐)을 누르면 개발 지속에 큰 힘이 된다고 팀이 밝히고 있습니다.
