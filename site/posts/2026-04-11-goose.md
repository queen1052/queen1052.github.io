## Goose란 무엇인가?

AI 에이전트 프레임워크는 많습니다. 하지만 코딩에만 특화된 것이 아닌, 리서치, 글쓰기, 자동화, 데이터 분석 등 모든 작업에 쓸 수 있는 **범용 AI 에이전트**는 드뭅니다.

**Goose**는 그런 범용 AI 에이전트입니다. 원래는 Block(Square의 모회사)이 개발했고, 현재는 Linux Foundation의 **Agentic AI Foundation(AAIF)** 소속으로 관리됩니다. macOS, Linux, Windows용 네이티브 데스크탑 앱, 풀 CLI, API를 모두 제공하며, Rust로 작성되어 성능과 이식성이 뛰어납니다.

GitHub 스타 41,000개 이상, Rust 58.6% + TypeScript 33.9%, Apache-2.0 라이선스, 448명의 기여자를 보유한 대형 오픈소스 프로젝트입니다.

## 주요 기능 한눈에 보기

Goose의 핵심 강점은 넓은 호환성과 확장성입니다:

- **15개 이상의 AI 제공자**: Anthropic, OpenAI, Google, Ollama, OpenRouter, Azure, Amazon Bedrock 등
- **기존 구독 활용**: Claude Max/Pro, ChatGPT Plus, Gemini 구독을 API 키 없이 ACP를 통해 사용
- **70개 이상의 확장**: Model Context Protocol(MCP) 오픈 표준으로 확장 연결
- **멀티 플랫폼**: macOS, Linux, Windows 네이티브 데스크탑 앱
- **오픈 거버넌스**: Linux Foundation의 AAIF 소속, 투명한 의사결정

## 범용 에이전트 설계: 코딩을 넘어서

기존 AI 코딩 어시스턴트들이 코드 생성에 집중하는 반면, Goose는 모든 작업에 사용 가능합니다.

**코딩**: 파일 읽기/쓰기, 테스트 실행, 오류 수정, PR 생성
**리서치**: 웹 검색, 문서 분석, 정보 종합
**자동화**: 파일 처리, API 호출, 데이터 파이프라인
**데이터 분석**: 데이터셋 분석, 시각화, 통계 계산
**글쓰기**: 문서 초안, 이메일, 보고서

이 모든 작업이 에이전트의 단일 인터페이스를 통해 이루어집니다.

## ACP와 MCP: 두 가지 확장 프로토콜

Goose는 두 가지 프로토콜로 기능을 확장합니다.

**ACP(Agent Communication Protocol)**: Claude Max/Pro, ChatGPT Plus, Gemini 등 기존 AI 구독을 API 키 없이 활용할 수 있게 합니다. 유료 구독이 있다면 별도 API 키 없이 Goose에서 해당 모델을 사용할 수 있습니다.

**MCP(Model Context Protocol)**: Anthropic이 주도하는 오픈 표준으로, 70개 이상의 확장 연결을 지원합니다. GitHub, Slack, Notion, 파일 시스템, 데이터베이스 등 다양한 서비스와 통합됩니다.

## AAIF: 오픈소스 AI 에이전트의 새 홈

Goose가 Linux Foundation의 AAIF에 합류한 것은 중요한 의미를 가집니다. AAIF(Agentic AI Foundation)는 오픈소스 AI 에이전트 생태계를 위한 중립적인 재단입니다. Linux Foundation의 거버넌스 모델은 투명한 의사결정, 기여자 보호, 지속 가능한 개발을 보장합니다.

## 기술 스택 및 아키텍처

- **언어**: Rust 58.6% (성능/이식성), TypeScript 33.9% (UI), Shell 1.9%, JavaScript 1.8%, Python 1.5%
- **라이선스**: Apache-2.0
- **플랫폼**: macOS, Linux, Windows
- **빌드 도구**: Cargo(Rust), pnpm(TypeScript)
- **v1.30.0** 최신 릴리스 (127개 릴리스)

## 설치 및 사용법

### 데스크탑 앱

[goose-docs.ai](https://goose-docs.ai/docs/getting-started/installation)에서 macOS, Linux, Windows용 앱을 다운로드할 수 있습니다.

### CLI 설치

```bash
curl -fsSL https://github.com/aaif-goose/goose/releases/download/stable/download_cli.sh | bash
```

### 빠른 시작

1. 앱 또는 CLI 설치
2. AI 제공자 선택 (Anthropic, OpenAI, Google 등)
3. API 키 설정 또는 ACP로 기존 구독 연결
4. 자연어로 작업 지시

커스텀 배포판 구성도 가능합니다. 미리 설정된 제공자, 확장, 브랜딩을 포함한 자체 Goose 배포판을 만들 수 있습니다(`CUSTOM_DISTROS.md` 참고).

Goose는 단순한 코딩 어시스턴트를 넘어선 진정한 범용 AI 에이전트입니다. Rust의 성능, 15개 이상의 AI 제공자 지원, 70개 이상의 MCP 확장, Linux Foundation의 견고한 거버넌스가 결합된 프로젝트입니다. Block에서 AAIF로의 이전은 이 프로젝트의 장기적인 지속 가능성을 보장합니다. 41,000여 스타와 448명의 기여자로 오픈소스 AI 에이전트 생태계의 핵심 프로젝트로 자리 잡은 Goose를 지금 바로 사용해보세요.
