## 소개

MiroFish는 haje01가 개발한 **집단 지성 시뮬레이션 예측 엔진**입니다. 프로젝트 이름 "MiroFish"는 미래(未來)를 예측하는 물고기 떼(집단 지성)에서 영감을 받은 것으로 보입니다. Python 57.9%, Vue 40.9%로 구현되어 있으며 AGPL-3.0 라이선스로 공개됩니다.

핵심 기술은 **GraphRAG + Zep Cloud** 조합입니다. GraphRAG는 지식을 단순한 벡터가 아닌 그래프 구조로 표현해 개념 간의 관계와 추론 경로를 유지합니다. Zep Cloud는 장기 기억 관리 플랫폼으로 에이전트의 대화 이력과 사실 기억을 지속적으로 유지합니다. 이 조합으로 여러 에이전트가 지식을 공유하며 집단적으로 예측하는 시뮬레이션을 구현합니다.

작동 방식은 간단합니다: 예측 대상(질문)을 입력하면, 여러 에이전트가 다양한 관점에서 독립적으로 분석하고 GraphRAG 지식 그래프를 통해 관련 정보를 참조합니다. 각 에이전트의 예측이 집계되어 최종 집단 예측이 만들어집니다. 이는 인간 전문가 패널의 델파이 방법론(Delphi Method)을 AI 에이전트로 구현한 것과 유사합니다.

## 주요 기능

- **멀티 에이전트 집단 예측**: 여러 에이전트가 독립적으로 분석하고 예측한 후, 가중치 평균 또는 앙상블로 최종 예측을 산출합니다. 단일 에이전트보다 다양성 있는 예측이 가능합니다.
- **GraphRAG 지식 베이스**: 지식을 그래프로 구조화해 개념 간의 관계를 명시적으로 표현합니다. 단순 벡터 검색보다 복잡한 추론 경로를 따라 관련 지식을 검색합니다.
- **Zep Cloud 장기 기억**: 에이전트의 대화 이력, 사실 기억, 이전 예측 결과를 Zep Cloud에 저장해 장기간 학습 효과를 유지합니다.
- **Vue 프론트엔드**: 직관적인 Vue 기반 UI로 예측 질문 입력, 에이전트 토론 과정 시각화, 최종 예측 결과 확인이 가능합니다.
- **예측 파이프라인**: 그래프 구축 → 환경 설정 → 시뮬레이션의 3단계 파이프라인으로 예측을 실행합니다.
- **커스텀 에이전트 역할**: 낙관론자, 비관론자, 중립 전문가, 악마의 변호인(Devil's Advocate) 등 다양한 관점의 에이전트 역할을 정의하고 시뮬레이션합니다.

### GraphRAG + Zep Cloud — 집단 지성의 기술

```
[예측 질문] "2025년 말 AI 에이전트 시장 규모는?"

그래프 구축:
  AI 에이전트 → [connected_to] → LLM
  AI 에이전트 → [market_trend] → 2024 성장률 (→ 156% YoY)
  AI 에이전트 → [key_player] → OpenAI, Anthropic, Google
  LLM → [enables] → 에이전트 자동화
  기업 채택 → [drives] → AI 에이전트 수요

멀티 에이전트 시뮬레이션:
  낙관론자 에이전트: "성장률 유지 시 $15B 예상"
  비관론자 에이전트: "버블 가능성, $8B 조정 예상"
  현실주의자: "기업 도입 속도 고려 시 $11B"
  
집계: 가중 평균 → "$11.3B (불확실성 ±$3B)"
```

### 3단계 예측 파이프라인

```bash
# 단계 1: 지식 그래프 구축
python build_graph.py \
  --topic "AI agent market 2025" \
  --sources web,papers,reports \
  --output graph.json

# 단계 2: 환경 및 에이전트 설정
python setup_env.py \
  --agents optimist,pessimist,realist,devils_advocate \
  --zep-key YOUR_ZEP_KEY \
  --graph graph.json

# 단계 3: 시뮬레이션 실행
python run_simulation.py \
  --question "2025년 말 AI 에이전트 시장 규모?" \
  --rounds 5 \
  --aggregate weighted_mean
```

## 설치 / 사용법

```bash
# 저장소 클론
git clone https://github.com/haje01/MiroFish
cd MiroFish

# 백엔드 설치
pip install -r requirements.txt

# 프론트엔드 설치
npm install

# 환경 변수 설정
cp .env.example .env
# ZEP_API_KEY, OPENAI_API_KEY 또는 다른 LLM API 설정

# 개발 서버 실행
# 백엔드
python -m uvicorn app.main:app --reload

# 프론트엔드
npm run dev
```

## 활용 사례 / 사용 시나리오

1. **기술 트렌드 예측**: AI, 블록체인, 양자 컴퓨팅 같은 기술 분야의 향후 발전 방향을 집단 에이전트 시뮬레이션으로 예측합니다. 여러 관점을 통합한 균형 잡힌 예측을 얻습니다.

2. **시장 조사 도구**: 신제품 출시나 시장 진입 결정을 위한 시장 규모와 성장률을 GraphRAG 기반으로 분석합니다. 에이전트들이 낙관적/비관적/현실적 시나리오를 각각 탐색합니다.

3. **정책 영향 분석**: 특정 정책이나 규제가 산업에 미치는 영향을 시뮬레이션합니다. 다양한 이해관계자 역할(기업, 소비자, 정부)의 에이전트가 각 관점에서 영향을 분석합니다.

## 결론

MiroFish는 집단 지성이라는 오래된 아이디어를 LLM 에이전트와 GraphRAG로 현대화한 흥미로운 프로젝트입니다. Zep Cloud의 장기 기억과 GraphRAG의 지식 구조화가 결합되어 단순한 LLM 예측보다 풍부하고 다양성 있는 결과를 제공합니다. 한국 개발자가 만든 프로젝트로, 복잡한 예측 문제를 집단 에이전트로 접근하는 독창적인 방법론이 인상적입니다.
