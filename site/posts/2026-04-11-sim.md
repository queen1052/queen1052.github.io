## 소개

**Sim**은 AI 에이전트를 구축하고, 배포하고, 오케스트레이션하기 위한 오픈소스 플랫폼입니다. [sim.ai](https://sim.ai)로 클라우드 버전을 무료로 사용할 수 있으며, 자체 호스팅도 완벽히 지원합니다.

1,000개 이상의 통합과 LLM을 연결하여 에이전트 워크플로우를 오케스트레이션하는 Sim은 캔버스 기반 비주얼 빌더, AI Copilot으로 노드 자동 생성, 벡터 데이터베이스 통합을 통한 RAG 등 현대적인 에이전트 인프라의 모든 요소를 갖추고 있습니다.

---

## 주요 기능

Sim의 핵심 기능들:

- **비주얼 워크플로우 빌더**: 캔버스에서 에이전트, 도구, 블록을 드래그앤드롭으로 연결
- **AI Copilot**: 자연어로 노드 생성, 오류 수정, 플로우 이터레이션
- **벡터 DB 통합**: 문서를 벡터 스토어에 업로드하여 RAG 에이전트 구현
- **1,000개 이상 통합**: Google Sheets, Drive, Calendar, GitHub 등
- **멀티 LLM 지원**: OpenAI, Anthropic, Gemini, DeepSeek 등
- **리얼타임 소켓**: Socket.io 기반 실시간 워크플로우 상태
- **백그라운드 잡**: Trigger.dev로 스케줄 및 웹훅 실행
- **원격 코드 실행**: E2B를 통한 안전한 코드 실행

---

## 핵심 기능 1: 비주얼 워크플로우 빌더 + Copilot

Sim의 가장 차별화된 특징은 **AI Copilot이 내장된 비주얼 빌더**입니다:

**워크플로우 빌더:**
- 캔버스에서 에이전트, LLM, 도구 블록을 시각적으로 연결
- 연결 조건 설정, 병렬 실행, 루프 패턴 지원
- 한 번 만든 워크플로우를 즉시 실행

**AI Copilot:**
- "이메일을 읽어서 CRM에 기록하는 에이전트 만들어줘"처럼 자연어로 노드 생성
- 실행 오류 발생 시 Copilot에게 수정 요청 가능
- 기존 플로우에 기능 추가도 자연어 지시로 처리

**Copilot 자체 호스팅 설정:**
```
COPILOT_API_KEY=sim.ai에서 생성한 키
```

## 핵심 기능 2: 지식 베이스와 RAG 통합

Sim은 벡터 스토어를 통해 에이전트가 특정 문서 기반으로 답변하는 RAG(검색 증강 생성) 패턴을 쉽게 구현합니다:

- 문서를 벡터 스토어에 업로드 (다양한 청킹 전략 지원)
  - Token, Sentence, Recursive, Regex 청커 제공
- 에이전트 블록에서 지식 베이스 연결
- 질문이 들어오면 자동으로 관련 청크를 검색해 답변에 사용

**지원 청킹 전략:**
- **Token 청킹**: 토큰 수 기반 균일 분할
- **Sentence 청킹**: 문장 경계 인식
- **Recursive 청킹**: 계층적 구분자 기반
- **Regex 청킹**: 커스텀 패턴 기반

## 설치 및 사용법

**가장 빠른 시작 (NPM):**
```bash
npx simstudio
# → http://localhost:3000
# Docker 실행 필요
```

**Docker Compose:**
```bash
git clone https://github.com/simstudioai/sim.git && cd sim
docker compose -f docker-compose.prod.yml up -d
# http://localhost:3000
```

**수동 설치 (개발자):**
```bash
git clone https://github.com/simstudioai/sim.git
cd sim && bun install
docker run --name simstudio-db -e POSTGRES_PASSWORD=pw \
  -e POSTGRES_DB=simstudio -p 5432:5432 -d pgvector/pgvector:pg17
cp apps/sim/.env.example apps/sim/.env
# 비밀 키 생성
perl -i -pe "s/your_encryption_key/$(openssl rand -hex 32)/" apps/sim/.env
cd packages/db && bun run db:migrate
bun run dev:full
```

## 마치며

Sim은 텍스트 기반 AI API 래퍼 수준에서 진정한 에이전트 오케스트레이션 플랫폼으로 도약하는 도구입니다. 2.7만 스타, 237개 릴리스, 55명의 기여자가 보여주는 활발한 커뮤니티와, 아파치 2.0 라이선스의 완전한 오픈소스가 Sim을 신뢰할 수 있는 선택으로 만듭니다. 노코드부터 풀스택 API까지 필요한 수준으로 사용할 수 있는 유연성이 Sim의 강점입니다.
