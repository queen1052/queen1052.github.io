## lark-cli란 무엇인가?

업무 자동화를 위해 Lark/Feishu API를 직접 호출하는 것은 번거롭습니다. 인증, 요청 포맷, 페이지네이션, 오류 처리... 이 모든 것을 개발자가 직접 다뤄야 합니다. **lark-cli**는 이 복잡함을 제거합니다.

공식 larksuite 팀이 유지관리하는 lark-cli는 단순한 CLI 도구를 넘어 **AI 에이전트 친화적 설계** 철학으로 만들어졌습니다. 모든 명령어가 실제 에이전트와 함께 테스트되었고, 간결한 파라미터, 스마트 기본값, 구조화된 출력으로 에이전트 호출 성공률을 최대화합니다. GitHub 스타 7,400개 이상으로 빠르게 성장 중입니다.

## 주요 기능 한눈에 보기

lark-cli가 커버하는 비즈니스 도메인은 방대합니다:

| 도메인 | 기능 |
|--------|------|
| 📅 캘린더 | 일정 조회, 이벤트 생성, 참가자 초대, 여유 시간 확인 |
| 💬 메신저 | 메시지 발송/답장, 그룹 채팅 관리, 이력 조회 |
| 📄 문서 | 문서 생성/읽기/수정/검색, 미디어 및 화이트보드 |
| 📁 드라이브 | 파일 업로드/다운로드, 댓글 관리 |
| 📊 Base | 테이블, 필드, 레코드, 뷰, 대시보드, 워크플로우 |
| 📈 스프레드시트 | 생성, 읽기, 쓰기, 추가, 검색, 내보내기 |
| ✅ 업무 | 생성, 조회, 업데이트, 완료, 하위 업무, 알림 |
| 📚 위키 | 지식 공간, 노드, 문서 관리 |
| 📧 메일 | 탐색, 검색, 읽기, 발송, 답장, 전달, 초안 |
| 🎥 회의 | 회의 기록 검색, 회의록 및 녹화 조회 |
| ✍️ 결재 | 결재 업무 조회, 승인/반려/이관 |

## AI 에이전트 스킬 시스템 심층 탐구

lark-cli의 가장 독특한 점은 20개의 AI 에이전트 스킬입니다. 이 스킬들은 에이전트가 Lark를 제로 설정으로 조작할 수 있게 해줍니다.

각 스킬은 도메인에 특화된 컨텍스트, 도구 설명, 사용 패턴을 포함합니다. `lark-shared` 스킬은 앱 설정, 인증, 아이덴티티 전환, 범위 관리 등을 담당하며 다른 모든 스킬에 자동으로 로드됩니다. 주요 스킬: lark-calendar, lark-im, lark-doc, lark-drive, lark-sheets, lark-base, lark-task, lark-mail, lark-contact, lark-wiki, lark-event 등.

스킬 설치는 npm 생태계와 통합되어 있습니다:
```bash
npx skills add larksuite/cli -y -g
```

## 3계층 명령어 시스템

lark-cli는 용도에 맞는 세 가지 레벨의 명령어를 제공합니다:

**1. 숏컷 명령어** (`+` 접두사): 인간과 AI 모두에 친화적, 스마트 기본값, 테이블 출력
```bash
lark-cli calendar +agenda
lark-cli im +messages-send --chat-id "oc_xxx" --text "안녕하세요"
lark-cli docs +create --title "주간 보고" --markdown "# 진행 현황\n- 기능 X 완료"
```

**2. API 명령어**: Lark OAPI 메타데이터에서 자동생성, 100개 이상 엔드포인트 매핑
```bash
lark-cli calendar calendars list
lark-cli calendar events instance_view --params '{"calendar_id":"primary",...}'
```

**3. Raw API 호출**: 2500개 이상 Lark Open Platform API에 직접 접근
```bash
lark-cli api GET /open-apis/calendar/v4/calendars
lark-cli api POST /open-apis/im/v1/messages --data '{"receive_id":"..."}'
```

## 기술 스택 및 아키텍처

- **언어**: Go 97.8% (빠른 실행 속도), JavaScript 1.9%
- **라이선스**: MIT (자유로운 사용)
- **설치**: npm 또는 소스 빌드
- **최소 의존성**: Node.js(npm/npx), Go v1.23+, Python 3
- **보안**: 입력 인젝션 방지, 터미널 출력 살균, OS 네이티브 키체인 자격증명 저장

## 설치 및 사용법

```bash
# npm으로 설치 (권장)
npm install -g @larksuite/cli

# 스킬 설치 (필수)
npx skills add larksuite/cli -y -g

# 앱 자격증명 설정 (최초 1회, 대화형 안내)
lark-cli config init

# 로그인 (--recommend로 자주 쓰는 범위 자동 선택)
lark-cli auth login --recommend

# 인증 상태 확인
lark-cli auth status

# 사용 시작
lark-cli calendar +agenda
```

AI 에이전트를 위한 논블로킹 로그인:
```bash
# 백그라운드 실행, 인증 URL을 추출해 사용자에게 전달
lark-cli auth login --domain calendar --no-wait
# 나중에 폴링 재개
lark-cli auth login --device-code <DEVICE_CODE>
```

출력 형식 제어: `--format json/pretty/table/ndjson/csv`, 드라이런: `--dry-run`, 전체 페이지네이션: `--page-all`

## 마치며

lark-cli는 Lark/Feishu 생태계를 CLI와 AI 에이전트에서 완전히 활용할 수 있게 해주는 공식 도구입니다. 3계층 명령어 시스템으로 간단한 조작부터 전체 API 커버까지 모든 수준의 사용자를 지원합니다. 20개의 AI 에이전트 스킬은 에이전트가 추가 설정 없이 바로 Lark를 조작할 수 있게 해줍니다. MIT 라이선스, 7,400여 스타, 50명의 기여자로 빠르게 성장하는 이 프로젝트는 기업 업무 자동화의 새로운 표준이 될 것입니다.
