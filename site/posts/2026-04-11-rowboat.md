## 소개

**Rowboat**는 YC(와이콤비네이터) 지원 스타트업 Rowboat Labs가 개발한 로컬 우선(local-first) AI 동료 앱입니다. 단순한 AI 챗봇을 넘어서, 이메일·미팅 노트·문서를 연결해 장기적으로 성장하는 지식 그래프를 구축하고 그 맥락으로 실질적인 업무를 수행하는 도구입니다.

Rowboat의 핵심은 **기억이 복리로 쌓인다**는 것입니다. 대부분의 AI 도구가 매번 맥락을 재구성하는 반면, Rowboat는 장기 지식을 유지합니다. 중요한 사항은 직접 편집하고 삭제할 수 있으며, 모든 데이터는 로컬 Markdown 파일로 저장됩니다.

---

## 주요 기능

Rowboat가 제공하는 기능들:

- **Gmail 통합**: 이메일에서 자동으로 맥락 구축
- **Google Calendar 동기화**: 미팅 전 관련 히스토리 자동 준비
- **Rowboat 미팅 노트 / Fireflies**: 미팅 기록에서 핵심 정보 추출
- **라이브 노트**: `@rowboat` 입력으로 자동 업데이트되는 살아있는 노트
- **PDF 슬라이드 생성**: 지식 그래프 기반 발표 자료 자동 생성
- **MCP 확장**: Exa(웹 검색), Twitter/X, ElevenLabs(음성), Slack 등 외부 도구 연동
- **로컬 모델 지원**: Ollama, LM Studio와 호환
- **음성 입출력**: Deepgram(음성 입력), ElevenLabs(음성 출력) 선택적 지원

## 핵심 기능 1: 장기 지식 그래프 구축

Rowboat가 다른 AI 도구와 근본적으로 다른 이유는 **컨텍스트가 소모되지 않고 축적**된다는 것입니다.

**일반 AI 도구의 한계:**
- 매번 문서나 이메일을 수동으로 공유해야 함
- 이전 대화의 맥락이 사라짐
- 관계와 패턴을 스스로 파악하지 못함

**Rowboat의 접근 방식:**
- Obsidian 호환 평문 Markdown 볼트로 저장
- 백링크(backlinks)로 관련 정보 연결
- 시간이 지날수록 맥락이 더 풍부해짐
- 직접 편집하고 검사하고 백업 가능

실제 활용 예시:
```
"Alex와의 미팅 준비해줘"
→ 과거 결정 사항, 열린 질문, 관련 이메일 스레드를 모아 간결한 브리핑 생성
```

## 핵심 기능 2: 라이브 노트와 실시간 모니터링

**라이브 노트**는 Rowboat의 독특한 기능입니다. 노트에 `@rowboat`를 입력하면 해당 노트가 자동으로 업데이트되는 살아있는 문서가 됩니다.

**활용 사례:**
- 경쟁사나 시장 트렌드를 X, Reddit, 뉴스에서 실시간 추적
- 특정 인물, 프로젝트, 거래 관련 커뮤니케이션 모니터링
- 관심 주제의 최신 요약 자동 유지

**수행 가능한 실제 작업들:**
- `다음 분기 로드맵 발표 자료 만들어줘` → 지식 그래프 기반 PDF 생성
- `Alex와의 미팅 브리핑해줘` → 관련 이메일, 메모, 결정사항 취합
- `지난주 액션 아이템 추적해줘` → 업무 팔로업 자동화
- 음성 메모 녹음 → 핵심 내용 자동 지식 그래프 반영

## 기술 스택 및 아키텍처

**언어 및 프레임워크:**
- TypeScript 96.7%
- Python (일부)
- CSS, JavaScript

**핵심 설계 원칙:**
- 로컬 우선: 모든 데이터를 로컬 Markdown으로 저장
- 독점 포맷 없음: 언제든지 데이터 이동 가능
- 모델 중립: Ollama, LM Studio, 모든 API 지원

## 설치 및 사용법

**다운로드 (권장):**
[https://www.rowboatlabs.com/downloads](https://www.rowboatlabs.com/downloads)에서 Mac/Windows/Linux용 최신 버전을 다운로드합니다.

**선택적 기능 설정:**

```bash
# Google 서비스 연동
# google-setup.md 참조

# 음성 입력 활성화
~/.rowboat/config/deepgram.json
# {"apiKey": "<deepgram-key>"}

# 음성 출력 활성화
~/.rowboat/config/elevenlabs.json
# {"apiKey": "<elevenlabs-key>"}

# 웹 검색 (Exa)
~/.rowboat/config/exa-search.json
# {"apiKey": "<exa-key>"}
```

## 마치며

Rowboat는 AI 어시스턴트의 근본적인 문제인 "기억 없음"을 해결합니다. 매번 맥락을 다시 설명해야 하는 피로감 없이, 시간이 갈수록 더 유능해지는 진정한 의미의 AI 동료를 원하는 사람들에게 Rowboat는 최선의 선택입니다. 10만 이상의 별을 받은 이 프로젝트는 YC 출신답게 실용성과 기술력을 동시에 갖추었습니다.
