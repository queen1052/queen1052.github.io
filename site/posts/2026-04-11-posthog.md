## PostHog란?

**PostHog**는 성공적인 제품을 만들기 위해 필요한 모든 분석·실험 도구를 하나의 플랫폼에 통합한 오픈소스 개발자 플랫폼입니다. GitHub에서 32,500개 이상의 스타를 획득한 이 프로젝트는 "PostHog is an all-in-one, open source platform for building successful products"라는 슬로건에 걸맞게, 파편화된 SaaS 구독 스택을 단일 셀프호스팅 인스턴스로 대체할 수 있습니다.

Python 50%, TypeScript 40.7%, Rust 7.1%로 구성된 모노레포 구조이며, MIT 라이선스로 공개되어 있습니다 (EE 디렉토리 제외). 463명의 기여자가 활발하게 개발 중입니다.

## 주요 기능

PostHog가 하나의 플랫폼에서 제공하는 기능 스위트를 살펴보겠습니다.

**제품 분석(Product Analytics)**: 자동 캡처 또는 수동 계측으로 사용자 행동 이벤트를 수집하고, 퍼널·코호트·경로 분석 및 SQL 쿼리로 데이터를 분석합니다.

**웹 애널리틱스(Web Analytics)**: Google Analytics와 유사한 대시보드로 웹 트래픽과 사용자 세션을 모니터링합니다. 전환율, 코어 웹 바이탈, 수익 지표를 손쉽게 파악할 수 있습니다.

**세션 리플레이(Session Replay)**: 실제 사용자가 웹사이트나 모바일 앱과 상호작용하는 녹화 영상을 재생하여 문제를 진단하고 UX를 이해합니다.

**피처 플래그(Feature Flags)**: 선택한 사용자나 코호트에 안전하게 기능을 단계적으로 롤아웃합니다. 코드를 다시 배포하지 않고도 기능을 켜고 끌 수 있습니다.

**실험(Experiments/A·B 테스트)**: 변경사항을 테스트하고 목표 지표에 미치는 통계적 영향을 측정합니다. 노코드 실험 설정도 지원합니다.

**에러 트래킹(Error Tracking)**: 에러를 추적하고 알림을 받으며 이슈를 해결합니다.

**설문조사(Surveys)**: 노코드 설문 템플릿 모음 또는 커스텀 설문 빌더로 원하는 무엇이든 질문합니다.

## 데이터 인프라 기능

PostHog는 단순한 분석 도구에 그치지 않고 데이터 인프라 역할도 합니다.

**데이터 웨어하우스(Data Warehouse)**: Stripe, HubSpot, 외부 데이터 웨어하우스 등 외부 도구의 데이터를 동기화하고 제품 데이터와 함께 쿼리할 수 있습니다.

**데이터 파이프라인(CDP)**: 수신 데이터에 커스텀 필터와 변환을 적용하고 25개 이상의 도구로 전송하거나 대용량 배치를 웨어하우스로 내보낼 수 있습니다.

**LLM 분석**: LLM 기반 앱의 트레이스, 생성 결과, 레이턴시, 비용을 캡처합니다.

**워크플로우 자동화**: 사용자에게 액션이나 메시지를 보내는 자동화 워크플로우를 생성합니다.

## 기술 스택

```
백엔드: Python (Django) + 50% 비중
프론트엔드: TypeScript/React + 40.7% 비중  
스트리밍/고성능: Rust + 7.1% 비중
데이터베이스: ClickHouse (이벤트 스토리지), PostgreSQL (메타데이터)
메시징: Kafka
캐싱: Redis
컨테이너: Docker + Kubernetes
```

ClickHouse를 이벤트 스토어로 사용함으로써 수십억 개의 이벤트도 빠르게 쿼리할 수 있습니다. Kafka는 대용량 이벤트 스트림을 안정적으로 처리하는 역할을 담당합니다.

SDK는 JavaScript, React, React Native, Next.js, Vue, iOS, Android, Flutter, Python, Node.js, PHP, Ruby, Go, .NET/C# 등 주요 언어/프레임워크를 모두 지원합니다.

## 설치 및 사용법

**PostHog Cloud (권장)**: 가장 빠른 시작 방법입니다.

```
https://us.posthog.com/signup 또는 https://eu.posthog.com/signup
```

무료 티어: 이벤트 월 100만 건, 리플레이 5,000건, 플래그 요청 100만 건, 예외 10만 건, 설문 1,500응답 무료.

**셀프호스팅 (고급)**: Docker Compose를 이용한 취미 배포:
```bash
git clone https://github.com/PostHog/posthog.git
cd posthog
docker compose -f docker-compose.hobby.yml up -d
```

**JavaScript 스니펫 설치**:
```html
<script>
  !function(t,e){/* PostHog snippet */}(window, 'phog_YOUR_API_KEY', 'https://us.i.posthog.com')
</script>
```

**이벤트 캡처**:
```javascript
posthog.capture('user_signed_up', {
  plan: 'pro',
  source: 'landing_page'
})

// 피처 플래그 확인
if (posthog.isFeatureEnabled('new-dashboard')) {
  showNewDashboard()
}
```

## 마치며

PostHog는 "왜 여러 SaaS 툴에 매달 수천 달러를 지불해야 하나?"라는 질문에 대한 명쾌한 답을 제시합니다. Mixpanel, Amplitude, Hotjar, LaunchDarkly, Optimizely의 기능을 하나의 오픈소스 플랫폼에서 무료로 혹은 셀프호스팅으로 사용할 수 있습니다. 463명의 활발한 기여자 커뮤니티와 완전한 소스 공개는 강력한 커스터마이징 가능성을 보장합니다. 제품 중심 성장(PLG) 전략을 추구하는 스타트업에게 특히 강력히 추천합니다.
