## 소개

Sleek UI는 luongnv89가 공개한 **브랜드 디자인 시스템 JSON 컬렉션**입니다. TypeScript 53.9%로 구현되어 있으며 Apache 2.0 라이선스로 공개됩니다. Stripe, GitHub, Linear, Vercel, Notion 등 55개 이상의 인기 브랜드와 디자인 시스템을 **shadcn/ui HSL 색상 형식**의 JSON 파일로 변환해 제공합니다.

핵심 아이디어는 간단합니다: "AI 에이전트나 개발자가 특정 브랜드의 디자인 시스템을 쉽게 가져다 쓸 수 있게 하자." 각 브랜드의 색상 팔레트, 타이포그래피, 컴포넌트 스타일을 구조화된 JSON으로 표현해 AI 코드 생성 도구에 주입하거나 `npx shadcn@latest init` 명령의 테마로 즉시 사용할 수 있습니다.

특히 AI 코딩 어시스턴트(Claude, Copilot)와의 시너지가 유용합니다. "Stripe 스타일로 랜딩 페이지를 만들어줘"라고 요청할 때 Sleek UI의 Stripe JSON을 컨텍스트로 제공하면, AI가 정확한 색상값과 디자인 토큰을 사용해 코드를 생성합니다.

## 주요 기능

- **55개+ 브랜드 디자인 시스템**: Stripe, GitHub, Linear, Vercel, Notion, Figma, Discord, Slack 등 유명 테크 브랜드의 디자인 시스템을 포함합니다. 지속적으로 업데이트됩니다.
- **shadcn/ui HSL 형식**: 모든 색상이 `hsl(220 14% 96%)` 형식으로 저장됩니다. shadcn/ui 기반 Next.js 프로젝트에 즉시 붙여넣기 가능합니다.
- **완전한 CSS 변수 세트**: `--background`, `--foreground`, `--primary`, `--secondary`, `--accent`, `--muted` 등 shadcn/ui의 전체 CSS 변수를 커버합니다.
- **다크모드 지원**: 라이트 테마와 다크 테마 색상을 모두 포함합니다. `:root`와 `.dark` 셀렉터에 맞는 변수를 각각 제공합니다.
- **AI 친화적 URL 패턴**: `https://sleek-ui.com/themes/stripe`처럼 예측 가능한 URL로 AI 에이전트가 프로그래매틱하게 접근할 수 있습니다.
- **TypeScript 타입 정의**: 디자인 토큰의 구조가 TypeScript 인터페이스로 정의되어 타입 안전한 방식으로 사용할 수 있습니다.

### JSON 구조 — 브랜드 디자인 토큰의 표준화

```json
{
  "name": "Stripe",
  "description": "Stripe의 클린하고 모던한 디자인 시스템",
  "light": {
    "--background": "0 0% 100%",
    "--foreground": "240 10% 3.9%",
    "--primary": "231 48% 48%",
    "--primary-foreground": "0 0% 98%",
    "--secondary": "240 4.8% 95.9%",
    "--accent": "240 4.8% 95.9%",
    "--muted": "240 4.8% 95.9%",
    "--border": "240 5.9% 90%",
    "--radius": "0.5rem"
  },
  "dark": {
    "--background": "240 10% 3.9%",
    "--foreground": "0 0% 98%",
    "--primary": "231 48% 58%"
    // ... 나머지 변수
  }
}
```

### shadcn/ui 통합 — 3단계 즉시 적용

포함된 브랜드 테마를 shadcn/ui 프로젝트에 적용하는 방법:

1. Sleek UI에서 원하는 브랜드 JSON 복사
2. 프로젝트의 `globals.css`에 붙여넣기
3. `shadcn/ui` 컴포넌트를 빌드하면 자동으로 브랜드 색상이 적용됨

별도의 테마 라이브러리나 커스텀 CSS 없이 shadcn/ui 기본 컴포넌트가 원하는 브랜드 스타일로 렌더링됩니다.

## 설치 / 사용법

```bash
# npm 패키지로 설치
npm install @sleek-ui/themes

# 또는 CDN으로 개별 테마 가져오기
curl https://sleek-ui.com/api/themes/stripe > stripe-theme.json

# Node.js에서 사용
import { themes } from '@sleek-ui/themes'
const stripeTheme = themes.stripe

# globals.css에 직접 붙여넣기
:root {
  --background: 0 0% 100%;
  --foreground: 240 10% 3.9%;
  --primary: 231 48% 48%;
  /* ... Sleek UI JSON에서 복사한 나머지 */
}

.dark {
  --background: 240 10% 3.9%;
  /* ... 다크 테마 변수 */
}
```

AI 에이전트와 함께 사용:
```
"Stripe 디자인 시스템(https://sleek-ui.com/themes/stripe)을 사용해서 
대시보드 컴포넌트를 만들어줘"
```

## 활용 사례 / 사용 시나리오

1. **AI 기반 UI 생성**: Claude, GPT에게 특정 브랜드 스타일로 UI를 생성해달라고 요청할 때 Sleek UI JSON을 컨텍스트로 제공합니다. AI가 정확한 HSL 값으로 코드를 생성합니다.

2. **빠른 프로토타이핑**: 고객사 브랜드 스타일을 적용한 프로토타입을 빠르게 만들 때, 유사한 브랜드의 Sleek UI 테마를 출발점으로 사용합니다. 처음부터 색상을 고르는 시간을 절약합니다.

3. **디자인 시스템 학습**: 인기 브랜드들이 어떻게 색상 팔레트, 다크모드, 컴포넌트 스타일을 구성했는지 JSON으로 비교 분석합니다. 디자이너와 개발자의 디자인 시스템 이해를 높입니다.

## 결론

Sleek UI는 단순하지만 매우 실용적인 아이디어를 잘 실행한 프로젝트입니다. 브랜드 디자인 시스템을 JSON으로 표준화해 AI 도구와의 통합을 수월하게 합니다. shadcn/ui 생태계와의 자연스러운 통합이 특히 Next.js 개발자에게 즉시 가치를 제공합니다. 55개 이상의 브랜드 컬렉션으로 다양한 프로젝트에서 재사용 가능합니다.
