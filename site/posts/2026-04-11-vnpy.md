## 소개

**VeighNa**(구 vn.py)는 39,200개 이상의 GitHub 스타를 보유한 중국의 대표적인 오픈소스 퀀트 트레이딩 플랫폼입니다. Python 99.4%로 구현되었으며 MIT 라이선스로 배포됩니다.

현재 버전 4.3.0에서는 중국 국내 선물 및 증권 브로커 30개 이상과의 직접 연결, LightGBM·Lasso·MLP 기반 AI 알파 팩터, CTA·스프레드·스크립트 전략, SQLite부터 DolphinDB·TDengine까지의 다양한 데이터베이스 지원을 갖추고 있습니다.

## 주요 기능

- **30+ 브로커 연결**: CTP, CFFEX 등 중국 주요 선물/증권 브로커와의 직접 API 연결
- **AI-Powered 전략**: LightGBM, Lasso, MLP 기반 Alpha158 팩터 라이브러리
- **다양한 전략 앱**: CTA(추세추종), 스프레드 트레이딩, 스크립트 기반 전략
- **다중 DB 지원**: SQLite, MySQL, PostgreSQL, MongoDB, DolphinDB, TDengine
- **VeighNa Studio**: 통합 GUI 트레이딩 환경
- **백테스팅**: 과거 데이터 기반 전략 성능 검증

## AI 알고 트레이딩 심층 분석

VeighNa의 AI 전략 모듈은 Alpha158 팩터 라이브러리를 기반으로 합니다. Alpha158은 158개의 기술적/통계적 알파 팩터로 구성된 표준 팩터 집합으로, 원래 Microsoft Research Asia에서 개발했습니다.

**LightGBM**: 트리 앙상블 기반 그래디언트 부스팅으로 비선형 팩터 관계 포착에 강점. 필드 내 피처 중요도 해석이 가능하여 어떤 팩터가 수익을 예측하는지 확인 가능합니다.

**Lasso 회귀**: 정규화를 통해 과적합을 방지하고 핵심 팩터를 자동 선택합니다. 해석성이 높아 전략의 논리적 근거를 설명하기 용이합니다.

**MLP(다층 퍼셉트론)**: 딥러닝 기반으로 복잡한 비선형 패턴을 학습합니다. 충분한 데이터가 있을 때 가장 강력하지만 과적합 위험도 높습니다.

## CTA 전략과 스프레드 트레이딩

CTA(Commodity Trading Advisor) 전략은 추세추종의 핵심입니다. VeighNa의 CTA 앱은 이동평균 교차, 볼린저밴드, ATR 기반 포지션 사이징 등 클래식 기법을 기본 제공하며, 사용자 정의 전략 코드를 작성할 수 있습니다.

스프레드 트레이딩 앱은 두 관련 자산 간의 가격 차이(스프레드)를 활용하는 통계적 차익거래 전략을 지원합니다. 선물 월물 간 롤 스프레드, 관련 종목 간 페어 트레이딩 등이 대표적인 활용 사례입니다.

전략 예시:
```python
from vnpy.app.cta_strategy import CtaTemplate, StopOrder, TickData, BarData, BarGenerator, ArrayManager

class DoubleMaStrategy(CtaTemplate):
    """이중 이동평균 전략"""
    fast_window = 10
    slow_window = 20
    
    def on_bar(self, bar: BarData):
        self.am.update_bar(bar)
        if not self.am.inited:
            return
        
        fast_ma = self.am.sma(self.fast_window)
        slow_ma = self.am.sma(self.slow_window)
        
        if fast_ma > slow_ma and self.pos == 0:
            self.buy(bar.close_price, 1)
        elif fast_ma < slow_ma and self.pos > 0:
            self.sell(bar.close_price, 1)
```
