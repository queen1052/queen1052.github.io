## 소개

[mcp-use](https://github.com/mcp-use/mcp-use)는 Model Context Protocol(MCP) 생태계를 위한 풀스택 개발 프레임워크입니다. Manufact 팀이 개발한 이 라이브러리는 TypeScript와 Python 양쪽에서 MCP 서버와 MCP 앱을 손쉽게 구축할 수 있도록 설계되었습니다.

GitHub에서 9.8k 스타를 기록하며 MCP 생태계의 핵심 개발 도구로 자리잡은 mcp-use는 단순히 MCP 서버를 만드는 것을 넘어, Claude, ChatGPT 등 다양한 AI 클라이언트에서 동작하는 인터랙티브 위젯(MCP 앱)까지 지원합니다. "한 번 작성, 어디서나 실행"이라는 철학을 MCP 생태계에 적용한 것입니다.

## 주요 기능

mcp-use는 MCP 개발 생명 주기 전반을 하나의 프레임워크로 커버합니다:

- **MCP 서버 구축**: TypeScript/Python SDK로 MCP 도구(Tool) 정의 및 서버 실행
- **MCP 앱(위젯)**: React 컴포넌트를 활용해 Claude, ChatGPT 등에서 동작하는 인터랙티브 UI 구성
- **내장 Inspector**: MCP 서버/앱 테스트 및 디버깅을 위한 웹 기반 도구 (서버 자동 포함)
- **Manufact 클라우드 배포**: GitHub 레포 연결만으로 프로덕션 배포. 관찰성, 메트릭, 로그, 브랜치 배포 지원
- **MCP 에이전트 & 클라이언트**: 풀 MCP 에이전트 및 클라이언트 구현 포함
- **MCP 프로토콜 준수**: 공식 MCP 적합성(Conformance) 테스트 통과

## 핵심 기능 심층 분석: MCP 앱과 위젯

mcp-use의 가장 혁신적인 기능은 **MCP 앱**입니다. 기존 MCP 서버는 텍스트 응답만 가능했지만, mcp-use의 위젯 시스템을 사용하면 AI 응답 안에 풍부한 인터랙티브 UI를 삽입할 수 있습니다.

서버 측에서 위젯을 정의하면:

```typescript
server.tool({
  name: "get-weather",
  schema: z.object({ city: z.string() }),
  widget: "weather-display", // resources/weather-display/widget.tsx 참조
}, async ({ city }) => {
  return widget({
    props: { city, temperature: 22, conditions: "Sunny" },
    message: `${city}의 날씨: 맑음, 22°C`,
  });
});
```

클라이언트(Claude, ChatGPT)는 텍스트 메시지 대신 React 컴포넌트로 렌더링된 날씨 카드를 표시합니다. `resources/` 폴더의 위젯 파일은 자동으로 발견되어 별도 등록 없이 동작합니다.

## 핵심 기능 심층 분석: 빠른 개발 경험

mcp-use는 개발자 경험을 최우선으로 설계되었습니다:

**TypeScript 빠른 시작:**
```bash
npx create-mcp-use-app@latest
```

**Python 빠른 시작:**
```bash
pip install mcp-use
```

`@mcp-use/cli`를 사용하면 코드 변경 시 서버 자동 재시작(Hot Reload)이 활성화됩니다. Inspector는 서버 실행과 동시에 `http://localhost:3000/inspector`에서 자동으로 접근 가능하여 별도 설정 없이 즉시 테스트할 수 있습니다.

코드 에이전트(Claude Code, Cursor 등) 사용자를 위한 mcp-use 스킬도 제공됩니다: `skills.sh/mcp-use/mcp-use/mcp-apps-builder`

## 기술 스택 및 아키텍처

mcp-use는 모노레포 구조로 다음 패키지들을 포함합니다:

**TypeScript 패키지:**
- `mcp-use`: 핵심 프레임워크 (MCP 서버, 앱, 에이전트)
- `@mcp-use/cli`: 핫 리로드 및 자동 Inspector 내장 빌드 도구
- `@mcp-use/inspector`: MCP 서버 웹 기반 디버거
- `create-mcp-use-app`: 프로젝트 스캐폴딩 도구

**Python 패키지:**
- `mcp-use`: 완전한 MCP 서버 및 에이전트 SDK

기술 스택: TypeScript (76.9%) + Python (18.1%). Manufact 클라우드에 배포 시 자동 TLS, 커스텀 도메인, 트래픽 모니터링을 제공합니다.

## 설치 및 사용법

**TypeScript - 첫 MCP 서버 생성:**
```typescript
import { MCPServer, text } from "mcp-use/server";
import { z } from "zod";

const server = new MCPServer({ name: "my-server", version: "1.0.0" });

server.tool({
  name: "get_weather",
  description: "도시 날씨 조회",
  schema: z.object({ city: z.string() }),
}, async ({ city }) => {
  return text(`온도: 22°C, 맑음, 위치: ${city}`);
});

await server.listen(3000);
// Inspector: http://localhost:3000/inspector
```

**프로덕션 배포:**
```bash
npx @mcp-use/cli login
npx @mcp-use/cli deploy
```

**사용 가능한 템플릿**: Chart Builder, Diagram Builder, Maps Explorer, Recipe Finder, File Manager 등 15개 이상의 즉시 배포 가능한 MCP 앱 템플릿 제공.

## 마치며

mcp-use는 MCP 개발을 단순한 API 서버 구축에서 풍부한 인터랙티브 AI 앱 개발로 격상시키는 프레임워크입니다. TypeScript/Python 동시 지원, 내장 Inspector, 원클릭 클라우드 배포의 조합은 MCP 생태계에서 가장 완성도 높은 개발 환경을 제공합니다. AI 도구를 만드는 개발자라면 mcp-use가 워크플로우의 핵심이 될 것입니다.
