## 소개

**ReClip**은 yt-dlp를 백엔드로 활용하는 경량 자체호스팅 동영상/오디오 다운로더입니다. GitHub에서 3,700개의 스타와 650개의 포크를 기록했으며, HTML 75.5%, Python 19.8%로 구성된 MIT 라이선스 프로젝트입니다.

전체 백엔드 코드가 약 150줄의 Python Flask로 구성되어 있어 누구나 쉽게 코드를 이해하고 자신의 환경에 맞게 수정할 수 있습니다. 프론트엔드는 단일 HTML 파일로 빌드 도구 없이 동작하는 바닐라 웹 UI입니다.

## 주요 기능

- **1000+ 사이트 지원**: yt-dlp가 지원하는 모든 사이트 (YouTube, TikTok, Instagram, Twitter/X, Reddit, Facebook, Vimeo, Twitch, Dailymotion, SoundCloud 등)
- **MP4/MP3 모드**: 비디오 또는 오디오만 추출해 다운로드
- **화질 선택**: 해상도/품질 옵션 선택 가능
- **일괄 다운로드**: 여러 URL을 한 번에 처리, 자동 URL 중복 제거
- **자동 저장**: 다운로드 완료 시 자동 저장 및 긴 파일명 자동 단축
- **Docker 지원**: 컨테이너 환경에서 의존성 격리 실행

## 기술적 단순함의 미학

ReClip의 강점은 역설적으로 '단순함'입니다. ~150줄의 Flask 백엔드는 다음 역할을 합니다:

1. 사용자로부터 URL 수신
2. yt-dlp를 subprocess로 호출하여 메타데이터(제목, 썸네일, 가용 화질) 조회
3. 사용자의 화질 선택 후 yt-dlp로 실제 다운로드
4. ffmpeg으로 포맷 변환(MP4→MP3 등)
5. 완성된 파일을 HTTP 응답으로 전달

비동기 작업 큐, 데이터베이스, 사용자 인증이 없는 완전 스테이트리스 설계입니다. 이 단순함 덕분에 보안 취약점이 적고, 유지보수가 쉬우며, 커스터마이징이 자유롭습니다.

## yt-dlp의 강력함

ReClip이 1000+ 사이트를 지원할 수 있는 것은 yt-dlp 덕분입니다. yt-dlp는 youtube-dl의 포크로, 더 활발한 업데이트와 더 많은 사이트 지원, 더 빠른 다운로드 속도를 제공합니다.

각 플랫폼의 콘텐츠 보호 우회, 인증이 필요한 콘텐츠(쿠키 기반), HLS/DASH 스트리밍 병합 등 복잡한 기능을 yt-dlp가 처리하므로, ReClip의 Flask 서버는 단순 래퍼 역할만 합니다.

## 기술 스택 및 아키텍처

- **언어**: HTML 75.5%, Python 19.8%
- **라이선스**: MIT
- **백엔드**: Python Flask (~150줄)
- **다운로드 엔진**: yt-dlp
- **미디어 처리**: ffmpeg
- **프론트엔드**: 바닐라 HTML/CSS/JavaScript (빌드 단계 없음)
- **의존성**: Flask, yt-dlp (Python 패키지 2개만)
- **포트**: 기본 8899

## 설치 및 사용법

```bash
# 시스템 의존성 설치
brew install yt-dlp ffmpeg    # macOS
# 또는: sudo apt install ffmpeg && pip install yt-dlp

# 저장소 클론
git clone https://github.com/averygan/reclip.git
cd reclip

# 실행
./reclip.sh
# → http://localhost:8899
```

Docker로 실행:
```bash
docker build -t reclip .
docker run -p 8899:8899 -v $(pwd)/downloads:/downloads reclip
```

사용 방법:
1. http://localhost:8899 접속
2. 비디오 URL 붙여넣기 (여러 줄로 여러 URL 가능)
3. MP4(비디오) 또는 MP3(오디오) 선택
4. Fetch 클릭 → 썸네일과 화질 옵션 확인
5. Download 또는 Download All 클릭

## 마치며

**ReClip**은 "150줄로 1000+ 사이트 다운로더"라는 명제를 현실로 보여주는 프로젝트입니다. 3,700 스타는 이 단순한 접근법이 많은 개발자와 사용자에게 실용적인 가치를 제공한다는 증거입니다. 복잡한 의존성 없이 Flask와 yt-dlp 두 패키지만으로 완전한 미디어 다운로더를 구현한 코드는, 좋은 소프트웨어가 반드시 복잡할 필요가 없다는 것을 보여줍니다. 개인 사용이나 팀 내 공유 다운로더 서버로 즉시 배포 가능합니다.
