## 소개

OpenSpace는 HKUDS(홍콩대 데이터 사이언스 그룹)가 공개한 **자율 진화 에이전트 플랫폼**입니다. GitHub에서 4,900개의 별을 기록하며 빠르게 주목받고 있습니다. 핵심 아이디어는 명확합니다: AI 에이전트가 실패로부터 배워 스스로 스킬(도구)을 진화시키도록 한다는 것입니다. 인간이 스킬을 수동으로 설계하거나 수정하는 대신, **에이전트가 스킬 진화 그래프를 구축하고 새로운 태스크를 만났을 때 자동으로 가장 적합한 스킬을 선택·개선**합니다.

OpenSpace의 실적 지표는 인상적입니다. GDPVal 경제 벤치마크에서 기준(baseline) 대비 **수입을 4.2배** 더 창출했으며, 자율 운영 과정에서 **165개의 스킬이 진화**했습니다. 스킬 진화 그래프는 스킬들 간의 상속 관계, 개선 이력, 성능 변화를 시각적으로 추적합니다.

더불어 HKUDS의 DeepTutor, TradingAgents 등 다른 프로젝트와의 시너지를 고려하면, OpenSpace는 범용 에이전트 백본 스킬 레이어로 발전할 가능성이 있습니다.

## 주요 기능

- **자율 스킬 진화**: 에이전트가 새로운 태스크를 실패하면, 기존 스킬 라이브러리에서 가장 유사한 스킬을 선택해 LLM 기반으로 변형·개선합니다. 성공한 스킬은 라이브러리에 추가되어 재사용됩니다.
- **스킬 진화 그래프**: 진화 이력을 DAG(방향성 비순환 그래프) 형태로 시각화합니다. 어떤 스킬이 어떤 스킬로부터 파생되었는지, 성능 변화 추이를 추적할 수 있습니다.
- **GDPVal 경제 벤치마크**: 현실적인 경제 활동(시장, 계약, 자원 배분)을 시뮬레이션하는 벤치마크로 에이전트 성능을 평가합니다. 학문적 NLP 태스크가 아닌 실제 비즈니스 능력을 측정합니다.
- **스킬 재사용 시스템**: 이전에 성공한 태스크와 유사한 새 태스크가 등장하면 기존 스킬을 재활용해 학습 속도와 비용을 동시에 줄입니다.
- **멀티 에이전트 협업**: 복잡한 태스크를 여러 전문화된 에이전트로 분해하고, 각 에이전트가 자신의 스킬 라이브러리를 독립적으로 진화시키면서 협력합니다.
- **저비용 운영**: 스킬 재사용 덕분에 같은 태스크를 매번 새로 추론하는 Instead 방식보다 API 비용이 현저히 감소합니다.

### 스킬 진화 메커니즘 — "실패가 학습이 된다"

기존 에이전트 프레임워크에서 에이전트가 태스크에 실패하면 단순히 오류를 반환합니다. OpenSpace는 실패를 **스킬 개선의 신호**로 활용합니다.

실패 발생 → 실패 분석 (어떤 스텝에서 왜 실패했는가) → 스킬 라이브러리 검색 (가장 유사한 기존 스킬 선택) → LLM 기반 스킬 변형 (실패 원인을 반영한 변형 생성) → 새 스킬 검증 → 성공 시 라이브러리 추가

165개의 자율 진화 스킬은 이 과정의 산물입니다.

### GDPVal 벤치마크 — 현실적 에이전트 측정

많은 에이전트 벤치마크가 텍스트 요약, 질의응답 같은 단일 태스크를 측정합니다. GDPVal은 에이전트가 **경제 생태계에서 다른 에이전트와 상호작용하며 부(경제적 가치)를 창출**하는 능력을 측정합니다. OpenSpace 에이전트는 이 벤치마크에서 기준선 대비 4.2배의 수입을 기록했습니다.

## 기술 스택 / 아키텍처

| 항목 | 내용 |
|------|------|
| 언어 | Python |
| 연구 기관 | HKUDS (홍콩대) |
| 벤치마크 | GDPVal Economic Benchmark |
| 진화된 스킬 | 165개 (자율) |
| 별 | 4,900+ |
| 관련 프로젝트 | DeepTutor, TradingAgents, OpenAgentSys |

아키텍처:  
`태스크 입력 → 스킬 라이브러리 검색 → 스킬 실행 → 성공: 라이브러리 강화 / 실패: 스킬 진화 → 결과 반환`

## 설치 / 사용법

```bash
# 저장소 클론
git clone https://github.com/HKUDS/OpenSpace
cd OpenSpace

# 의존성 설치
pip install -r requirements.txt

# 환경 설정 (.env 파일 생성)
cp .env.example .env
# OPENAI_API_KEY 또는 호환 LLM API 키 설정

# 기본 에이전트 실행
python run_agent.py --task "your task description"

# 스킬 라이브러리 확인
python show_skills.py --graph  # 진화 그래프 시각화

# GDPVal 벤치마크 실행
python benchmark.py --gdpval
```

## 활용 사례 / 사용 시나리오

1. **장기 자율 에이전트 구축**: 사람의 개입 없이 장기간 실행되어야 하는 에이전트에 적합합니다. 스스로 학습하고 진화하므로, 초기에는 실패하더라도 점차 성공률이 높아집니다.

2. **스킬 공유 생태계**: 여러 에이전트가 공용 스킬 라이브러리를 공유하면, A 에이전트가 학습한 스킬을 B 에이전트가 즉시 활용할 수 있습니다. 조직 내 AI 에이전트 팀이 집단 지성으로 성장합니다.

3. **에이전트 비용 최적화**: 같은 유형의 태스크를 반복 실행하는 경우, 첫 실행에서 스킬을 학습한 후 후속 실행에서 재사용해 LLM API 호출 비용을 크게 줄일 수 있습니다.

## 결론

OpenSpace는 에이전트 자율화의 핵심 과제인 "지속적 학습"을 우아하게 해결합니다. 스킬 진화 그래프, GDPVal 벤치마크, 4.2배 성능 향상이라는 구체적인 성과가 아이디어의 실용성을 입증합니다. 현재 Python 기반이며, HKUDS의 다른 프로젝트들(DeepTutor, TradingAgents)과 통합하면 더 강력한 AI 에이전트 스택을 구성할 수 있습니다.
