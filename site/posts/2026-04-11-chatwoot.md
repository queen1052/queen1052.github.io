## 소개

[Chatwoot](https://github.com/chatwoot/chatwoot)는 Intercom, Zendesk, Salesforce Service Cloud의 오픈소스 대안을 표방하는 현대적 고객 지원 플랫폼입니다. 28.4k 스타를 기록한 이 플랫폼은 라이브 채팅, 이메일, Facebook, Instagram, Twitter, WhatsApp, Telegram, Line, SMS 등 모든 채널의 고객 대화를 하나의 강력한 인박스로 통합합니다.

Ruby on Rails 백엔드와 Vue.js 프론트엔드로 구축되어 강력하면서도 자체 호스팅이 용이합니다. MIT 라이선스(일부 기능 제외)로 제공되며, Heroku, DigitalOcean, Docker 등 다양한 방식으로 배포할 수 있습니다.

## 주요 기능

Chatwoot는 현대 고객 지원 팀이 필요한 모든 도구를 제공합니다:

**Captain AI 에이전트:**
- 일반 질문 자동 응답
- 에이전트 워크로드 감소
- 복잡한 문제는 인간 에이전트에게 자동 에스컬레이션

**옴니채널 지원 데스크:**
- 웹사이트 라이브 채팅, 이메일, WhatsApp, Telegram, Facebook, Instagram, Twitter, Line, SMS 통합
- 모든 채널의 대화를 하나의 인터페이스에서 관리

**협업 & 생산성:**
- 비공개 노트와 `@멘션`으로 팀 내 토론
- 레이블로 대화 분류 및 정리
- 키보드 단축키 및 커맨드 바
- 캔드 응답(자주 묻는 질문 빠른 답변)
- 자동 배정 및 팀 자동화

## 핵심 기능 심층 분석: Captain AI 에이전트

Chatwoot v4의 핵심 기능인 Captain은 고객 지원에 특화된 AI 에이전트입니다. 단순한 챗봇이 아니라 실제 비즈니스 컨텍스트를 이해하는 지능형 지원 자동화 시스템을 제공합니다.

**작동 방식:**
1. 고객 대화가 시작되면 Captain이 질문을 분석
2. 내부 지식 베이스, FAQ, 헬프센터 아티클을 검색
3. 답변 가능한 경우 자동으로 정확한 응답 제공
4. 복잡하거나 감정적인 대화는 인간 에이전트에게 라우팅
5. 에이전트는 Captain이 처리하지 못한 복잡한 케이스에만 집중

이를 통해 팀은 반복적인 일반 문의에서 해방되어 실제로 전문성이 필요한 복잡한 문제에 집중할 수 있습니다.

## 핵심 기능 심층 분석: 고객 데이터 & 보고서

Chatwoot는 단순한 채팅 도구를 넘어 풍부한 고객 인텔리전스를 제공합니다:

**고객 데이터 관리:**
- 연락처 관리: 상세 프로필과 상호작용 이력
- 연락처 세그먼트: 타겟 커뮤니케이션을 위한 세분화
- 커스텀 속성: 추가 고객 데이터 저장
- 사전 채팅 폼: 대화 시작 전 사용자 정보 수집

**통합:**
- Slack: Slack에서 직접 대화 관리
- Dialogflow: 챗봇 자동화
- Shopify: 고객 주문 정보 확인
- Google Translate: 실시간 메시지 번역
- Linear: 티켓 생성 및 관리

**보고서:**
- 대화, 에이전트, 인박스, 레이블, 팀별 리포트
- CSAT(고객 만족도) 리포트
- 실시간 모니터링 뷰

## 기술 스택 및 아키텍처

- **백엔드**: Ruby on Rails (46.2%)
- **프론트엔드**: Vue.js (28.0%) + JavaScript (23.0%)
- **데이터베이스**: PostgreSQL (pgvector 포함)
- **실시간**: ActionCable (WebSocket)
- **배포**: Docker, Heroku, DigitalOcean Kubernetes

## 설치 및 사용법

**Heroku 원클릭 배포:**
[Deploy](https://heroku.com/deploy?template=https://github.com/chatwoot/chatwoot/tree/master) 버튼 클릭 후 환경 변수 설정.

**Docker Compose:**
```bash
git clone https://github.com/chatwoot/chatwoot.git
cd chatwoot
cp .env.example .env
docker compose up -d
```

**DigitalOcean Kubernetes:**
마켓플레이스에서 원클릭 배포.

환경 변수 상세 설정은 chatwoot.com/docs/environment-variables 참조.

## 마치며

Chatwoot는 28.4k 스타와 331명의 컨트리뷰터가 증명하는 성숙한 오픈소스 고객 지원 플랫폼입니다. 월 수백 달러의 Intercom이나 Zendesk 구독 없이 동등한 수준의 기능을 자체 호스팅으로 운영할 수 있습니다. Captain AI 에이전트의 추가로 반복적인 지원 업무의 자동화가 가능해져, 지원 팀의 생산성을 획기적으로 높일 수 있습니다. 고객 지원 인프라를 완전히 소유하고 싶은 기업에게 Chatwoot는 가장 완성도 높은 선택입니다.
