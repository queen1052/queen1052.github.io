## LocalAI란?

**LocalAI**는 "The open-source AI engine"이라는 타이틀에 걸맞게, LLM, 비전, 음성, 이미지, 영상 생성을 어떤 하드웨어에서도 GPU 없이 실행할 수 있는 무료 오픈소스 AI 엔진입니다. GitHub 스타 45,200개를 보유한 이 프로젝트는 Ettore Di Giacinto가 만들고 유지 관리하며 189명의 기여자가 함께 개발합니다.

**핵심 가치**: OpenAI, Anthropic, ElevenLabs API와 완전 호환되므로 기존 OpenAI SDK 코드를 그대로 사용하면서 백엔드만 로컬로 전환할 수 있습니다. 데이터는 절대 인프라 밖을 벗어나지 않습니다.

Go 69.9%, JavaScript 11.9%, HTML 7%, Python 5.8%로 구성되어 있으며 MIT 라이선스로 공개되어 있습니다.

## 주요 기능

LocalAI의 방대한 기능 세트를 살펴보겠습니다.

**35+ 백엔드 지원**: llama.cpp, vLLM, transformers, whisper.cpp, diffusers, MLX, MLX-VLM 등 35개 이상의 백엔드를 지원하며 백엔드 갤러리에서 온디맨드로 설치·제거할 수 있습니다.

**다양한 AI 작업**:
- 텍스트 생성 (LLM 추론)
- 텍스트-오디오 변환 (TTS)
- 오디오-텍스트 변환 (Whisper ASR)
- 이미지 생성 (Diffusers, Stable Diffusion)
- 임베딩 생성
- 비전 API (GPT-4 Vision 호환)
- Realtime API (Speech-to-Speech)
- 객체 탐지
- 리랭커 API

**빌트인 AI 에이전트**: 도구 사용, RAG, MCP, 스킬을 갖춘 자율 AI 에이전트를 기본 제공하며 Agent Hub에서 에이전트를 검색할 수 있습니다.

## 하드웨어 지원

LocalAI는 다양한 가속기를 지원하는 것이 가장 큰 장점 중 하나입니다.

| 하드웨어 | 지원 백엔드 |
|---------|-----------|
| NVIDIA GPU | CUDA 12/13 |
| AMD GPU | ROCm (HIP) |
| Intel GPU | oneAPI/SYCL |
| Apple Silicon | Metal (MLX) |
| NVIDIA Jetson | L4T CUDA |
| Vulkan GPU | Vulkan |
| CPU only | CPU 추론 |

NVIDIA, AMD, Intel, Apple Silicon 모두 하드웨어 가속을 지원하며, GPU가 없는 환경에서도 CPU 전용으로 동작합니다.

## 아키텍처

LocalAI는 2025년 7월 모든 백엔드를 메인 바이너리 외부로 이전하는 경량화·모듈화 아키텍처 리팩토링을 완료했습니다.

**백엔드 갤러리**: 백엔드를 OCI 이미지로 패키징하여 온디맨드로 설치·제거할 수 있습니다. 불필요한 백엔드를 설치하지 않아 바이너리 크기를 최소화합니다.

**분산 모드(v4 신기능)**: PostgreSQL + NATS를 사용한 수평 확장 분산 모드를 지원합니다. 여러 인스턴스에 걸쳐 모델 추론을 분산할 수 있습니다.

**P2P 추론**: libp2p 기반 P2P 아키텍처로 여러 노드에 걸쳐 모델을 분산 실행할 수 있습니다.

최근 React 프론트엔드로 UI가 재구성되었으며, WebRTC 지원, MLX-distributed via P2P와 RDMA, MCP 앱·클라이언트 지원 등이 추가되었습니다.

## 설치 및 사용법

**macOS**: DMG 파일을 다운로드하거나 Docker 사용:

```bash
# NVIDIA GPU
docker run -ti --name local-ai -p 8080:8080 --gpus all \
  localai/localai:latest-aio-gpu-nvidia-cuda-12

# CPU only
docker run -ti --name local-ai -p 8080:8080 \
  localai/localai:latest-aio-cpu
```

**모델 로드**:
```bash
# Hugging Face에서
local-ai run huggingface://TheBloke/phi-2-GGUF/phi-2.Q8_0.gguf

# Ollama OCI 레지스트리에서
local-ai run ollama://gemma:2b

# OCI 레지스트리에서
local-ai run oci://localai/phi-2:latest
```

**OpenAI SDK 호환 사용**:
```python
import openai

client = openai.OpenAI(
  base_url="http://localhost:8080/v1",
  api_key="dummy"  # LocalAI는 API 키를 검증하지 않음 (기본값)
)

response = client.chat.completions.create(
  model="gpt-4",
  messages=[{"role": "user", "content": "Hello!"}]
)
print(response.choices[0].message.content)
```

## 마치며

LocalAI는 AI 인프라의 "소유권"을 되찾아주는 프로젝트입니다. OpenAI API 비용이 부담스럽거나, 데이터 프라이버시가 중요하거나, 인터넷 연결 없이 AI를 사용해야 하는 환경이라면 LocalAI는 최선의 선택입니다. 35개 이상의 백엔드, 모든 주요 하드웨어 지원, 자율 AI 에이전트 내장, OpenAI 완전 호환이라는 강점으로 45,000개 이상의 스타를 받으며 커뮤니티의 강력한 지지를 받고 있습니다.
