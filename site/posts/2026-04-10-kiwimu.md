## 소개

Kiwimu는 Open330이 개발한 **위키 기반 지식 학습 플랫폼**입니다. 위키피디아 문서를 단순히 읽는 것을 넘어, 문서들 간의 연결 관계를 D3.js 지식 그래프로 시각화하고, 학습 단계별 커리큘럼을 자동 생성하며, KaTeX로 수식을 아름답게 렌더링합니다. Mermaid 다이어그램 지원도 내장되어 있어 복잡한 개념의 시각적 이해를 돕습니다.

Bun과 TypeScript로 구현된 현대적인 스택은 빠른 실행 속도를 보장합니다. GitHub Pages나 Vercel에 단 몇 분 만에 배포할 수 있어 개인 지식 베이스나 팀 학습 허브로 즉시 활용 가능합니다. 접근성, 다크모드, 모바일 반응형 UI를 기본 지원합니다.

프로젝트명 "Kiwimu"는 한국어 단어와 영어의 조합으로, "키위"처럼 작지만 영양 가득한 지식을 제공한다는 의미를 담고 있습니다. 수식이 많은 수학, 물리학, 알고리즘 관련 위키 문서를 학습할 때 특히 강점을 발휘합니다.

## 주요 기능

- **D3.js 지식 그래프 시각화**: 위키 문서들 간의 링크 관계를 인터랙티브 그래프로 표시합니다. 노드를 클릭하면 해당 문서로 이동하고, 연결 강도에 따라 엣지 굵기가 조절됩니다.
- **단계별 학습 커리큘럼**: Ingest → Phase1 → Phase2 → Phase2.5 → Phase3의 5단계 학습 경로를 자동 생성합니다. 기초 개념부터 심화 내용까지 순서에 맞게 학습합니다.
- **퀴즈 자동 생성**: 학습한 위키 문서를 기반으로 이해도 확인 퀴즈를 자동 생성합니다. 객관식, 빈칸 채우기, 개방형 질문을 포함합니다.
- **KaTeX 수식 렌더링**: LaTeX 수식을 아름답고 정확하게 렌더링합니다. 수학, 물리학, 통계학 문서의 수식을 완벽하게 지원합니다.
- **Mermaid 다이어그램**: 위키 문서에 포함된 또는 자동 생성된 Mermaid 다이어그램을 렌더링합니다. 프로세스 흐름, 클래스 다이어그램, 시퀀스 다이어그램을 시각화합니다.
- **대시보드 & 진행 추적**: 학습 진행도, 퀴즈 점수, 탐색한 지식 그래프를 대시보드에서 확인합니다.

### 5단계 학습 경로 — 체계적 지식 구축

```
Ingest   →  위키 문서 수집 및 파싱, 지식 그래프 구축
Phase 1  →  핵심 개념 파악 (주요 정의, 핵심 용어)
Phase 2  →  심화 이해 (세부 메커니즘, 수식 해석)
Phase 2.5→  연결 강화 (관련 문서 탐색, 연결 고리 발견)
Phase 3  →  통합 평가 (퀴즈, 지식 지도 완성, 요약 작성)
```

이 구조는 단순 독서가 아닌 **능동적 학습(active recall)**을 지원합니다. Phase 2.5에서 지식 그래프를 탐색하며 개념 간 연결을 발견하는 과정이 기억 정착에 특히 효과적입니다.

### D3.js 지식 그래프 — 위키의 숨겨진 구조 발견

위키피디아 문서는 수많은 내부 링크로 연결된 거대한 지식 네트워크입니다. Kiwimu의 D3.js 그래프 시각화는 이 네트워크에서 학습 중인 주제 주변의 연결 구조를 보여줍니다. 중심 개념에서 가까운 문서(직접 연결), 간접 연결, 고립된 섬 등 지식의 구조를 한눈에 파악할 수 있습니다. 그래프를 탐색하면서 예상치 못한 지식의 연결고리를 발견하는 "아하!" 경험을 제공합니다.

| 항목 | 내용 |
|------|------|
| 런타임 | Bun |
| 언어 | TypeScript |
| 그래프 | D3.js |
| 수식 | KaTeX |
| 다이어그램 | Mermaid |
| 배포 | GitHub Pages, Vercel |
| UI | 다크모드, 모바일, 접근성 |

## 설치 / 사용법

```bash
# 저장소 클론
git clone https://github.com/Open330/kiwimu
cd kiwimu

# Bun 설치 (없는 경우)
curl -fsSL https://bun.sh/install | bash

# 의존성 설치
bun install

# 개발 서버 시작
bun dev

# 프로덕션 빌드
bun run build

# GitHub Pages 배포
bun run deploy
```

위키 문서 추가:
```bash
# 위키피디아 문서 URL 로 지식 그래프 구축
bun run ingest "https://ko.wikipedia.org/wiki/강화학습"
```

## 활용 사례 / 사용 시나리오

1. **수학/과학 자기 학습**: KaTeX 수식 렌더링과 Mermaid 다이어그램 덕분에 수식이 많은 수학, 물리학, 통계학 위키 문서를 아름답게 읽고 학습합니다. 5단계 학습 경로로 개념을 체계적으로 쌓습니다.

2. **개인 지식 관리(PKM)**: GitHub Pages에 배포해 자신만의 지식 허브를 구축합니다. 위키 문서를 큐레이션하고 코멘트를 추가해 개인화된 학습 자료를 만듭니다.

3. **스터디 그룹 활용**: Vercel에 배포하고 팀에 공유합니다. 공통 학습 주제의 지식 그래프를 함께 탐색하고, 퀴즈로 이해도를 서로 확인합니다.

## 결론

Kiwimu는 위키피디아를 수동적 독서 도구에서 능동적 학습 플랫폼으로 전환합니다. D3.js 지식 그래프, KaTeX 수식, Mermaid 다이어그램의 조합이 특히 이공계 학습자에게 강력한 도구를 제공합니다. Bun + TypeScript의 현대적 스택으로 빠르고 배포가 쉽습니다. 개인 학습이나 팀 스터디에 즉시 활용할 수 있는 완성도 높은 프로젝트입니다.
