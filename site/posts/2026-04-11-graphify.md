## 소개

**graphify**는 코드, 문서, PDF, 이미지, 영상 파일을 포함하는 모든 폴더를 쿼리 가능한 지식 그래프로 변환하는 AI 코딩 어시스턴트 스킬입니다. Claude Code, Codex, OpenCode, Cursor, Gemini CLI, GitHub Copilot CLI, Aider, OpenClaw, Factory Droid, Trae 등 주요 AI 코딩 어시스턴트를 지원합니다.

핵심 가치 제안: Andrej Karpathy의 `/raw` 폴더처럼 논문, 트윗, 스크린샷, 노트가 뒤섞인 코퍼스를 다룰 때, 원본 파일을 직접 읽는 것 대비 **71.5배 적은 토큰**으로 동일한 답을 얻을 수 있습니다.

---

## 주요 기능

graphify의 주요 기능:

- **멀티모달 지원**: 코드(20개 언어), 문서(MD, PDF), 이미지, 영상, 오디오, YouTube URL
- **3단계 추출 파이프라인**: AST 파싱 → 음성 전사 → Claude 서브에이전트 추출
- **Leiden 커뮤니티 감지**: 임베딩 없이 그래프 위상만으로 클러스터링
- **신뢰도 태깅**: EXTRACTED / INFERRED / AMBIGUOUS 관계 분류
- **다양한 내보내기**: HTML 인터랙티브, JSON, SVG, GraphML, Neo4j, Obsidian
- **MCP 서버**: 구조화된 그래프 접근 API (`query_graph`, `get_node`, `get_neighbors`)
- **증분 업데이트**: SHA256 캐시로 변경된 파일만 재처리
- **항상-켜짐 통합**: Claude Code, Cursor 등에 PreToolUse 훅 설치

---

## 핵심 기능 1: 3단계 추출 파이프라인

graphify는 파일 유형에 따라 세 가지 다른 방식으로 지식을 추출합니다:

**1. AST 파싱 (코드 파일)**
- tree-sitter로 소스코드에서 클래스, 함수, 임포트, 콜 그래프, 독스트링 추출
- LLM 없이 로컬에서 처리 → 코드 내용이 외부에 전송되지 않음
- 지원 언어: Python, JS, TS, Go, Rust, Java, C, C++, Ruby, C#, Kotlin, Scala, PHP, Swift, Lua, Zig, PowerShell, Elixir, Objective-C, Julia

**2. 음성 전사 (영상/오디오)**
- faster-whisper로 로컬 전사 → 오디오가 절대 외부 전송 없음
- 코퍼스 god nodes에서 도출된 도메인 인식 프롬프트 사용
- YouTube URL 직접 추가: `yt-dlp`로 오디오만 다운로드 후 전사

**3. Claude 서브에이전트 (문서/이미지/전사)**
- 병렬 서브에이전트로 개념, 관계, 설계 근거 추출
- 하이퍼엣지: 3개 이상 노드를 연결하는 관계 표현
- EXTRACTED vs INFERRED 투명한 관계 분류

## 핵심 기능 2: 항상-켜짐 어시스턴트 통합

설치 후 가장 강력한 기능은 AI 코딩 어시스턴트에 지식 그래프를 자동 통합하는 것입니다:

```bash
# Claude Code에 통합
graphify claude install
# → CLAUDE.md 섹션 + PreToolUse 훅 설치

# Cursor에 통합
graphify cursor install
# → .cursor/rules/graphify.mdc (alwaysApply: true)

# Codex에 통합
graphify codex install
# → AGENTS.md + .codex/hooks.json PreToolUse 훅
```

**하는 일:**
1. 모든 Glob/Grep 호출 전 훅이 발동
2. 지식 그래프가 존재하면 `GRAPH_REPORT.md` 요약을 어시스턴트에 주입
3. 어시스턴트가 파일을 하나하나 grep하는 대신 그래프를 통해 탐색

결과: 코드베이스 이해 속도 대폭 향상, 아키텍처 결정의 "왜"를 즉시 파악

## 기술 스택 및 아키텍처

**언어:** Python 100%
**핵심 라이브러리:**
- NetworkX: 그래프 저장 및 조작
- graspologic (Leiden): 커뮤니티 감지
- tree-sitter: 코드 AST 파싱
- vis.js: 인터랙티브 HTML 시각화
- faster-whisper: 로컬 음성 전사

**설치:** `pip install graphifyy` (PyPI 공식 패키지명은 `graphifyy`)

---

## 설치 및 사용법

```bash
pip install graphifyy && graphify install

# Claude Code에서
/graphify .

# 특정 폴더
/graphify ./raw --mode deep

# 변경된 파일만 업데이트
/graphify ./raw --update

# YouTube 영상 추가
/graphify add https://youtube.com/...

# 직접 쿼리
graphify query "attention과 optimizer를 연결하는 것은?"
```

**출력 파일:**
- `graphify-out/graph.html` — 인터랙티브 그래프 (클릭, 검색, 커뮤니티 필터)
- `graphify-out/GRAPH_REPORT.md` — god nodes, 놀라운 연결, 제안 질문
- `graphify-out/graph.json` — 지속적 그래프

## 마치며

graphify는 AI 코딩 어시스턴트가 코드베이스를 이해하는 방식을 근본적으로 바꿉니다. 파일을 하나하나 읽는 대신 구조화된 지식 그래프를 통해 탐색함으로써, 더 빠르고 정확한 답변을 더 적은 비용으로 얻을 수 있습니다. 2만 별 이상을 받은 이 프로젝트는 대형 코드베이스를 다루는 모든 개발자에게 필수 도구가 될 잠재력을 가지고 있습니다.
