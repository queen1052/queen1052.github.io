## 소개

**Sherlock**은 사용자명 하나로 400개 이상의 소셜 미디어 및 온라인 플랫폼에서 계정을 찾아주는 OSINT(오픈소스 인텔리전스) 도구입니다. GitHub에서 80,800개 이상의 스타를 획득한 Python 기반 프로젝트로, MIT 라이선스로 배포됩니다.

보안 연구자, 디지털 포렌식 전문가, 개인 프라이버시 감사를 위해 널리 사용됩니다. 자신의 계정이 어떤 플랫폼에 노출되어 있는지 확인하거나, 보안 조사를 위해 특정 사용자명의 존재 여부를 확인하는 데 활용됩니다.

## 주요 기능

- **400+ 플랫폼 지원**: Twitter/X, Instagram, Facebook, GitHub, Reddit, LinkedIn 등 주요 SNS 및 틈새 플랫폼
- **CSV/XLSX 출력**: 검색 결과를 스프레드시트 형식으로 저장
- **Tor 지원**: 익명성을 위한 Tor 네트워크 통한 검색
- **프록시 지원**: HTTP/SOCKS 프록시를 통한 IP 우회
- **NSFW 사이트 선택적 포함**: 기본값 제외, 옵션으로 포함 가능
- **Docker 지원**: 컨테이너 환경에서 격리된 실행

## 동작 원리 심층 분석

Sherlock은 각 소셜 플랫폼에 대해 `존재 시 응답`과 `부재 시 응답`을 사전에 정의한 데이터베이스를 활용합니다. 검색 시 각 플랫폼에 HTTP 요청을 보내고 응답 패턴을 분석하여 계정 존재 여부를 판단합니다.

지원하는 탐지 방법은 크게 두 가지입니다. 첫 번째는 HTTP 상태 코드 확인으로, 404(없음)와 200(있음)을 구분합니다. 두 번째는 응답 텍스트 패턴 매칭으로, 일부 플랫폼은 항상 200을 반환하지만 "사용자를 찾을 수 없습니다" 같은 텍스트를 포함하는 경우를 처리합니다.

비동기 HTTP 요청으로 수백 개 플랫폼을 동시에 조회하므로, 전체 스캔이 통상 수 초 내에 완료됩니다.

## 활용 사례와 윤리적 고려

**합법적 활용 사례:**
- 자신의 온라인 풋프린트 파악 및 정리
- 보안 연구 및 취약점 분석
- 디지털 포렌식 조사 (법적 권한 있는 경우)
- 브랜드 보호: 자신의 브랜드명이 다른 플랫폼에 무단 등록되었는지 확인

**주의사항:**
- 타인의 계정을 동의 없이 추적하는 것은 많은 국가에서 법적 문제가 될 수 있습니다
- 결과는 항상 100% 정확하지 않을 수 있으므로 추가 검증이 필요합니다
- 플랫폼의 이용약관 준수가 필요합니다

## 기술 스택 및 아키텍처

- **언어**: Python 97.3%
- **라이선스**: MIT
- **HTTP 클라이언트**: aiohttp (비동기)
- **프록시**: SOCKS4/5, HTTP 프록시 지원
- **익명성**: Tor(SOCKS5 through localhost:9050)
- **출력 형식**: TXT, CSV, XLSX, JSON
- **플랫폼 데이터**: data.json (지원 사이트 정의)

## 설치 및 사용법

```bash
# pipx로 설치 (권장)
pipx install sherlock-project

# 기본 검색
sherlock username

# 다중 사용자명 검색
sherlock user1 user2 user3

# CSV 출력
sherlock username --csv

# XLSX 출력
sherlock username --xlsx

# Tor 사용
sherlock username --tor

# 프록시 사용
sherlock username --proxy http://127.0.0.1:8080

# NSFW 사이트 포함
sherlock username --nsfw

# Docker로 실행
docker run theyahya/sherlock sherlock username
```

## 마치며

**Sherlock**은 80.8k 스타가 증명하는 OSINT 분야의 필수 도구입니다. 400개 이상의 플랫폼을 수 초 만에 스캔하는 능력은 개인 프라이버시 감사에서 전문 보안 조사까지 폭넓은 활용을 가능하게 합니다. Tor와 프록시 지원으로 익명 조사도 지원하며, CSV/XLSX 출력으로 대규모 조사에서도 결과를 체계적으로 관리할 수 있습니다. 단, 항상 윤리적이고 합법적인 방법으로만 사용해야 한다는 점을 잊지 마세요.
