## 소개: 토크나이저를 버린 TTS의 혁신

텍스트를 음성으로 변환할 때 대부분의 TTS 시스템은 텍스트를 이산 토큰으로 변환하는 단계를 거친다. **VoxCPM2**는 이 토크나이저 단계를 완전히 제거하고, 확산 자기회귀(Diffusion Autoregressive) 아키텍처로 연속적인 음성 표현을 직접 생성한다.

**VoxCPM**은 OpenBMB(칭화대학교 자연언어처리 연구실)가 개발한 토크나이저 없는 TTS 시스템이다. GitHub **9.2k stars**, Python 100%, Apache-2.0 라이선스로 공개되어 있다. ICLR 2026에 채택된 연구 결과물이며, VoxCPM2는 현재 권장 릴리즈다.

18명의 기여자, 13개 릴리즈를 통해 꾸준히 발전하고 있으며, voxcpm.readthedocs.io에 완전한 문서가 제공된다.

## 주요 기능: 4가지 음성 생성 모드

**30개 언어 다국어 지원**이 VoxCPM2의 가장 강력한 기능이다. 언어 태그 없이 입력 텍스트 자체로 언어를 자동 감지하고 합성한다. 내부 30개 언어 ASR 벤치마크에서 우수한 성능을 보인다.

**음성 디자인(Voice Design)**은 참조 오디오 없이 자연어 설명만으로 완전히 새로운 목소리를 만드는 기능이다. 성별, 나이, 톤, 감정, 말하기 속도를 텍스트로 지정한다:

```python
wav = model.generate(
    text="(젊은 여성, 부드럽고 달콤한 목소리)안녕하세요, VoxCPM2에 오신 것을 환영합니다!",
    cfg_value=2.0,
    inference_timesteps=10,
)
```

텍스트 시작 부분의 괄호 안 설명이 음성 특성을 지정한다. 참조 오디오 녹음이 필요 없다는 점이 혁신적이다.

**제어 가능한 음성 복제(Controllable Cloning)**는 짧은 참조 클립으로 목소리를 복제하되, 감정·속도·표현을 스타일 가이던스로 조정할 수 있다. 원본 음색을 유지하면서 다른 스타일을 적용한다.

**얼티밋 복제(Ultimate Cloning)**는 참조 오디오와 그 트랜스크립트를 함께 제공해 모든 발화 뉘앙스(음색, 리듬, 감정, 스타일)를 완벽히 재현한다.

## 아키텍처 심층 분석

VoxCPM2는 4단계 파이프라인으로 작동한다: **LocEnc → TSLM → RALM → LocDiT**. 전체 과정이 AudioVAE V2의 잠재 공간에서 이루어지며 이산 토크나이저를 완전히 우회한다.

- **LocEnc(Local Encoder)**: 입력 텍스트의 지역적 특성 인코딩
- **TSLM(Token-Sentence Level Model)**: 문장 수준의 언어 모델링
- **RALM(Residual Attention Language Model)**: 잔차 기반 자기회귀 언어 모델
- **LocDiT(Local Diffusion Transformer)**: 확산 기반 고해상도 오디오 생성

이 아키텍처로 48kHz 네이티브 오디오 출력을 달성한다. RTX 4090에서 실시간 배율(RTF) ~0.30을 기록하고, Nano-vLLM으로 서빙하면 ~0.13까지 낮아진다.

**모델 버전 비교**:

| 버전 | 파라미터 | 샘플레이트 | 언어 | 음성 디자인 |
|------|---------|-----------|------|-----------|
| VoxCPM2 | 2B | 48kHz | 30개 | ✅ |
| VoxCPM1.5 | 0.6B | 44.1kHz | 2개(중/영) | ❌ |
| VoxCPM1 | 0.5B | 16kHz | 2개(중/영) | ❌ |

## 성능 벤치마크 및 파인튜닝

**Seed-TTS-eval**: WER(단어 오류율, 낮을수록 좋음)과 SIM(화자 유사도, 높을수록 좋음) 기준으로 최첨단 성능을 보인다.

**InstructTTSEval**: 음성 디자인 명령어 준수 평가에서 높은 점수를 기록한다. 자연어 설명으로 창조된 음성의 품질이 검증되었다.

**파인튜닝** 지원이 강력하다. SFT(전체 파인튜닝)와 LoRA(파라미터 효율적 파인튜닝) 모두 지원한다. WebUI로 훈련과 추론을 동시에 관리할 수 있다:

```bash
python lora_ft_webui.py   # http://localhost:7860에서 WebUI 열기
```

커스텀 화자 데이터로 파인튜닝하면 특정 목소리에 특화된 모델을 만들 수 있다. 파인튜닝 가이드는 voxcpm.readthedocs.io에서 상세히 제공된다.

## 에코시스템 및 배포

거대한 에코시스템이 구축되어 있다. **Nano-vLLM**: 고처리량 GPU 서빙(RTF ~0.13). **VoxCPM.cpp**: GGML/GGUF 형식으로 CPU, CUDA, Vulkan 추론. **VoxCPM-ONNX**: CPU 추론용 ONNX 익스포트. **VoxCPMANE**: Apple Neural Engine 백엔드. **voxcpm_rs**: Rust 재구현. **ComfyUI-VoxCPM**: 노드 기반 워크플로우.

**프로덕션 배포**(Nano-vLLM 사용):

```python
from nanovllm_voxcpm import VoxCPM
import numpy as np, soundfile as sf

server = VoxCPM.from_pretrained(model="/path/to/VoxCPM", devices=[0])
chunks = list(server.generate(target_text="안녕하세요, VoxCPM입니다!"))
sf.write("out.wav", np.concatenate(chunks), 48000)
server.stop()
```

동시 요청 배치 처리와 FastAPI HTTP 서버를 지원한다.

## 설치 및 사용법

**빠른 설치** (Python ≥ 3.10, <3.13, PyTorch ≥ 2.5.0, CUDA ≥ 12.0):

```bash
pip install voxcpm
```

**기본 TTS 사용**:

```python
from voxcpm import VoxCPM
import soundfile as sf

model = VoxCPM.from_pretrained("openbmb/VoxCPM2", load_denoiser=False)

wav = model.generate(
    text="VoxCPM2는 현재 권장되는 다국어 음성 합성 시스템입니다.",
    cfg_value=2.0,
    inference_timesteps=10,
)
sf.write("demo.wav", wav, model.tts_model.sample_rate)
```

**CLI 사용**:

```bash
# 음성 디자인 (참조 오디오 불필요)
voxcpm design \
  --text "VoxCPM2는 스튜디오 품질의 다국어 음성 합성을 제공합니다." \
  --output out.wav

# 도움말
voxcpm --help
```

**웹 데모** 로컬 실행:

```bash
python app.py --port 8808
# http://localhost:8808에서 접근
```

Hugging Face Spaces에서 온라인 데모도 사용 가능하다: `huggingface.co/spaces/OpenBMB/VoxCPM-Demo`.

## 마치며

VoxCPM2는 TTS 기술의 새로운 방향을 제시한다. 토크나이저 없이 연속 잠재 공간에서 직접 음성을 생성하는 접근법은 더 자연스럽고 표현력 있는 음성 합성을 가능하게 한다. 30개 언어 지원과 자연어만으로 새 목소리를 창조하는 음성 디자인 기능은 특히 혁신적이다.

단, 음성 복제 기술의 남용(사기, 사칭, 허위 정보)은 엄격히 금지되며, AI 생성 콘텐츠임을 명확히 표시하도록 권고한다. 연구, 교육, 콘텐츠 창작에 책임감 있게 활용한다면 강력한 도구가 될 것이다.
