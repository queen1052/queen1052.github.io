## TimesFM이란 무엇인가?

시계열 예측은 수십 년간 독립적인 모델 학습이 필요한 영역이었습니다. 판매 예측은 판매 모델로, 주가 예측은 주가 모델로, 날씨 예측은 날씨 모델로 따로따로 학습시켜야 했습니다. **TimesFM(Time Series Foundation Model)**은 이 패러다임을 바꿉니다.

Google Research가 개발한 TimesFM은 다양한 시계열 데이터로 사전학습된 파운데이션 모델입니다. LLM이 텍스트를 이해하듯 TimesFM은 시계열 패턴을 이해합니다. ICML 2024에서 "A decoder-only foundation model for time-series forecasting"으로 발표되었으며, BigQuery의 공식 Google 제품으로도 채택되었습니다. 현재 GitHub 스타 1만 6천여 개를 기록하고 있습니다.

## 주요 기능 한눈에 보기

TimesFM의 핵심 강점은 제로샷 예측 능력입니다. 특정 도메인 데이터 없이도 새로운 시계열에 대한 예측이 가능합니다.

- **제로샷 예측**: 추가 학습 없이 새로운 시계열 데이터 예측
- **디코더 전용 아키텍처**: GPT 스타일로 시계열 패턴 모델링
- **긴 컨텍스트**: TimesFM 2.5에서 최대 16k 컨텍스트 지원
- **분위수 예측**: 옵션 30M 분위수 헤드로 1k 지평선까지 연속적 분위수 예측
- **공변량 지원**: XReg를 통한 공변량(covariates) 지원 복구
- **BigQuery 통합**: 공식 Google 클라우드 제품으로 이용 가능

## TimesFM 2.5: 최신 버전의 혁신

TimesFM은 지속적으로 발전해왔습니다. 최신 버전 2.5는 이전 버전 대비 상당한 개선을 보여줍니다.

**TimesFM 2.5 vs 2.0 비교:**
- 파라미터 수: 500M → 200M (60% 감소로 더 빠른 추론)
- 컨텍스트 길이: 2,048 → 16,384 (8배 확장)
- 연속적 분위수 예측 추가: 30M 옵션 헤드로 1k 지평선까지
- frequency 인디케이터 제거: 더 단순한 API
- 새로운 예측 플래그 추가

Flax 기반 버전(더 빠른 추론)도 개발 중이며, 더 많은 문서와 노트북이 지속적으로 추가되고 있습니다.

## 디코더 전용 아키텍처 심층 탐구

TimesFM이 기존 ARIMA, Prophet, LSTM 대비 우수한 성능을 보이는 이유는 아키텍처에 있습니다.

기존 시계열 모델은 각 시리즈에 대해 독립적으로 학습되는 반면, TimesFM은 수백만 개의 실제 시계열 데이터(합성 및 실제)로 사전학습됩니다. 디코더 전용 구조는 GPT가 텍스트를 예측하는 것처럼 시계열의 다음 값을 예측합니다. 이 방식은 다양한 주기성(일별, 주별, 월별), 트렌드, 계절성 패턴을 학습한 범용 표현을 만들어냅니다.

Hugging Face 컬렉션([google/timesfm-release](https://huggingface.co/collections/google/timesfm-release-66e4be5fdb56e960c1e482a6))에서 모든 체크포인트를 다운로드할 수 있습니다.

## 기술 스택 및 지원 환경

- **언어**: Python 68.3%, HTML 22.7%(노트북), Jupyter Notebook 8.6%
- **백엔드**: PyTorch 또는 JAX(Flax) 선택 가능
- **라이선스**: Apache-2.0
- **패키지 관리**: `uv` 권장
- **하드웨어**: CPU, GPU, TPU, Apple Silicon 지원
- **클라우드**: Google BigQuery 공식 통합

## 설치 및 사용법

```bash
# 저장소 클론
git clone https://github.com/google-research/timesfm.git
cd timesfm

# 가상 환경 생성 및 활성화
uv venv
source .venv/bin/activate

# PyTorch로 설치
uv pip install -e .[torch]

# Flax로 설치 (선택)
uv pip install -e .[flax]

# XReg 공변량 지원 포함
uv pip install -e .[xreg]
```

### 코드 예시

```python
import torch
import numpy as np
import timesfm

torch.set_float32_matmul_precision("high")

model = timesfm.TimesFM_2p5_200M_torch.from_pretrained(
    "google/timesfm-2.5-200m-pytorch"
)

model.compile(
    timesfm.ForecastConfig(
        max_context=1024,
        max_horizon=256,
        normalize_inputs=True,
        use_continuous_quantile_head=True,
    )
)

point_forecast, quantile_forecast = model.forecast(
    horizon=12,
    inputs=[
        np.linspace(0, 1, 100),     # 첫 번째 시계열
        np.sin(np.linspace(0, 20, 67)),  # 두 번째 시계열
    ],
)
# point_forecast.shape: (2, 12)
# quantile_forecast.shape: (2, 12, 10) — 평균 + 10th~90th 분위수
```

## 마치며

TimesFM은 시계열 예측의 새로운 패러다임을 제시합니다. Google Research의 엄격한 연구를 기반으로 ICML 2024에서 검증받았고, BigQuery라는 실제 상용 서비스에도 통합되었습니다. 200M 파라미터라는 적절한 크기로 실용적인 배포가 가능하며, 16k 컨텍스트와 연속 분위수 예측으로 복잡한 시계열도 다룰 수 있습니다. 판매 예측, 금융 데이터 분석, 에너지 수요 예측, 이상 탐지 등 다양한 도메인에서 사전학습된 파운데이션 모델의 위력을 경험해보세요.
