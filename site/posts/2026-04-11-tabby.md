## 소개

[Tabby](https://github.com/TabbyML/tabby)는 GitHub Copilot의 오픈소스 온프레미스 대안을 표방하는 자체 호스팅 AI 코딩 어시스턴트입니다. TabbyML 팀이 개발하고 Rust로 구축된 이 프로젝트는 GitHub에서 33.4k 스타를 기록하며 기업 환경에서의 AI 코딩 지원 분야에서 강력한 위치를 확보했습니다.

별도의 DBMS나 클라우드 서비스 없이 단일 컨테이너로 실행되며, OpenAPI 인터페이스로 기존 인프라와 쉽게 통합됩니다. 소비자급 GPU에서도 동작해 비용 효율적인 AI 코딩 지원이 가능합니다.

## 주요 기능

Tabby는 코딩 어시스턴트 기능을 넘어 팀 전체의 컨텍스트를 이해하는 지능형 플랫폼으로 진화하고 있습니다:

- **인라인 코드 완성**: VS Code, JetBrains IDE 등에서 실시간 코드 자동 완성
- **Chat 사이드 패널**: `@` 메뉴로 코드베이스 컨텍스트 참조
- **Answer Engine**: 레포지토리와 문서를 인덱싱해 팀의 코딩 질문 답변
- **Pages**: Answer Engine 대화를 영속적인 공유 가능 페이지로 변환
- **GitLab MR 인덱싱**: GitLab 머지 리퀘스트를 컨텍스트로 활용
- **REST API 문서 인덱싱**: 자체 문서를 추가해 Tabby 지식 확장
- **소비자 GPU 지원**: NVIDIA 4GB VRAM부터 동작. AMD ROCm 지원

## 핵심 기능 심층 분석: 1분 만에 실행

Tabby의 가장 큰 강점 중 하나는 간편한 시작입니다:

```bash
docker run -it \
  --gpus all -p 8080:8080 -v $HOME/.tabby:/data \
  tabbyml/tabby \
  serve --model StarCoder-1B --device cuda \
  --chat-model Qwen2-1.5B-Instruct
```

이 명령 하나로 StarCoder-1B 완성 모델과 Qwen2-1.5B-Instruct 채팅 모델을 동시에 실행하는 Tabby 서버가 시작됩니다. GPU 없이 CPU 모드로도 동작하며, 클라우드 IDE(Google Colab 등)에서도 무료로 체험할 수 있습니다.

강력한 모델 생태계도 장점입니다: StarCoder, DeepSeek Coder, Qwen2.5-Coder 등 코딩 특화 모델들을 지원하며, 기업의 보안 요구사항에 맞게 외부 API 없이 완전히 사내에서 운영할 수 있습니다.

## 핵심 기능 심층 분석: 팀 지식 인덱싱과 Answer Engine

Tabby v0.29+에서 강화된 Answer Engine은 팀 전체의 지식을 Tabby에 통합합니다:

**지원 컨텍스트 소스:**
- Git 레포지토리 코드
- Confluence, GitHub/GitLab Wiki 등 문서
- REST API 문서 (Swagger/OpenAPI)
- GitLab 머지 리퀘스트 (v0.30+)

개발자가 "우리 API에서 인증은 어떻게 처리해?"라고 물으면, Tabby는 실제 코드베이스와 내부 문서를 검색해 정확한 답변을 제공합니다. 이를 Pages로 저장하면 팀 전체가 공유하는 영속적인 내부 지식 문서가 됩니다.

## 기술 스택 및 아키텍처

- **언어**: Rust (92.9%) - 성능, 메모리 안전성
- **머신러닝**: Python (4.5%) - 모델 서빙 레이어
- **추론 엔진**: llama.cpp (ggml-org) 기반
- **IDE 플러그인**: VS Code (TypeScript), IntelliJ/JetBrains (Kotlin)
- **컨테이너**: Docker 이미지 제공 (CUDA, ROCm 변형)

아키텍처 특징: DB 불필요, 단일 바이너리 배포. OpenAPI 스펙 완전 준수로 기존 개발 툴체인과 완벽한 통합.

## 설치 및 사용법

**Docker로 빠른 시작 (CUDA):**
```bash
docker run -it --gpus all -p 8080:8080 \
  -v $HOME/.tabby:/data \
  tabbyml/tabby \
  serve --model StarCoder-1B --device cuda
```

**VS Code 익스텐션 설치:**
VS Code 마켓플레이스에서 "Tabby" 검색 후 설치. 설정에서 서버 URL(`http://localhost:8080`)을 입력하면 자동 완성이 즉시 활성화됩니다.

**소스에서 빌드:**
```bash
git clone --recurse-submodules https://github.com/TabbyML/tabby
cd tabby
# Rust 환경 설정 후
cargo build
```

## 마치며

Tabby는 기업 환경에서 AI 코딩 어시스턴트를 도입하면서 데이터 보안과 비용 효율성을 모두 추구하는 팀에게 최고의 선택입니다. Rust로 구축된 고성능 코어, 소비자급 GPU 지원, 풍부한 팀 지식 인덱싱 기능은 Tabby를 단순한 Copilot 대안을 넘어 팀의 집단 지식을 활용하는 지능형 개발 파트너로 만들고 있습니다. 33.4k 스타와 249개의 릴리즈가 증명하는 활발한 개발 속도도 큰 장점입니다.
