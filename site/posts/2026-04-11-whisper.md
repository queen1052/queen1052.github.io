## Whisper란?

OpenAI가 2022년 공개한 **Whisper**는 대규모 약한 지도학습(weak supervision)으로 학습된 범용 음성 인식 모델입니다. GitHub에서 97,500개 이상의 스타를 받은 이 프로젝트는 단순한 음성→텍스트 변환을 넘어 다국어 음성 인식, 음성 번역, 언어 식별, 음성 활동 감지(VAD)를 하나의 모델로 수행합니다.

Transformer 기반 시퀀스-투-시퀀스 아키텍처를 채택했으며, 인터넷에서 수집한 680,000시간 규모의 다양한 오디오-텍스트 쌍으로 학습되었습니다. MIT 라이선스로 공개되어 상업적 활용도 자유롭습니다.

## 주요 기능

Whisper가 제공하는 핵심 기능들을 살펴보겠습니다.

**다국어 지원**: 99개 이상의 언어로 훈련되었으며, 영어부터 일본어·힌디어·아랍어까지 광범위한 언어에서 경쟁력 있는 인식 성능을 보입니다. 라틴계 언어에서는 특히 우수한 결과를 냅니다.

**멀티태스크 처리**: 음성 인식(ASR), 음성 번역(Speech-to-English Translation), 언어 식별, 타임스탬프 생성을 단일 모델로 수행합니다. 전통적인 파이프라인이 각각 별도 모델을 요구하는 것을 획기적으로 단순화했습니다.

**다양한 모델 크기**: tiny(39M)부터 large-v3(1550M), turbo(809M)까지 6가지 크기를 제공하여 메모리와 속도를 트레이드오프할 수 있습니다.

**CLI 및 Python API**: 커맨드라인에서 `whisper audio.mp3 --model turbo` 한 줄로, 또는 Python 코드에서 직접 임베드하여 사용할 수 있습니다.

## 모델 크기별 성능 비교

Whisper의 모델은 크기와 속도 간의 절충점을 명확히 보여줍니다.

| 모델 | 파라미터 수 | VRAM | 상대속도(large 대비) |
|------|------------|------|---------------------|
| tiny | 39M | ~1GB | ~10x |
| base | 74M | ~1GB | ~7x |
| small | 244M | ~2GB | ~4x |
| medium | 769M | ~5GB | ~2x |
| large | 1550M | ~10GB | 1x |
| turbo | 809M | ~6GB | ~8x |

**turbo** 모델은 large-v3를 최적화하여 대략 8배 빠른 속도에 미미한 정확도 하락만 있어 영어 전사에 가장 추천됩니다. 단, turbo는 번역 작업에 적합하지 않으므로 음성 번역이 필요하면 medium 이상을 사용해야 합니다.

`.en` 전용 모델(tiny.en, base.en 등)은 영어 전용 애플리케이션에서 다국어 모델보다 더 나은 성능을 발휘합니다.

## 멀티태스크 학습 아키텍처

Whisper의 가장 혁신적인 측면은 **멀티태스크 학습 포맷**입니다.

동일한 Transformer 인코더-디코더 모델이 여러 작업을 수행할 수 있도록 특수 토큰(task specifier)을 활용합니다. 예를 들어, `<|transcribe|>` 토큰은 음성 인식, `<|translate|>` 토큰은 영어 번역, 언어 코드 토큰(`<|ko|>`, `<|ja|>` 등)은 언어 지정에 사용됩니다.

30초 슬라이딩 윈도우로 오디오를 처리하며, 각 윈도우에서 자기회귀 방식으로 출력을 생성합니다. 이 접근법은 단일 모델이 전통적인 음성 처리 파이프라인의 여러 단계를 대체할 수 있게 해줍니다.

인코더는 로그-멜 스펙트로그램을 입력받고, 디코더는 예측할 토큰 시퀀스를 출력합니다. 타임스탬프 예측도 같은 시퀀스 내에서 처리됩니다.

## 기술 스택 및 요구사항

**언어**: Python 100%

**핵심 의존성**:
- PyTorch (1.10.1 이상)
- tiktoken (빠른 토크나이저)
- ffmpeg (오디오 파일 처리)
- Python 3.8~3.11

**하드웨어**: GPU가 없어도 실행 가능하나 large 모델은 CPU에서 매우 느림. turbo/medium 모델은 Colab T4 수준에서 실시간~준실시간으로 동작합니다.

**지원 오디오 형식**: flac, mp3, mp4, mpeg, mpga, m4a, ogg, wav, webm (ffmpeg가 처리 가능한 모든 형식)

## 설치 및 사용법

**설치**:
```bash
pip install -U openai-whisper
# ffmpeg 설치 (Ubuntu/Debian)
sudo apt update && sudo apt install ffmpeg
```

**CLI 사용법**:
```bash
# 영어 음성 전사 (turbo 모델)
whisper audio.mp3 --model turbo

# 한국어 음성 전사
whisper korean_speech.wav --language Korean --model medium

# 외국어 음성을 영어로 번역
whisper japanese.wav --model medium --language Japanese --task translate
```

**Python API**:
```python
import whisper

model = whisper.load_model("turbo")
result = model.transcribe("audio.mp3")
print(result["text"])

# 언어 감지
audio = whisper.load_audio("audio.mp3")
audio = whisper.pad_or_trim(audio)
mel = whisper.log_mel_spectrogram(audio, n_mels=model.dims.n_mels).to(model.device)
_, probs = model.detect_language(mel)
print(f"감지된 언어: {max(probs, key=probs.get)}")
```

## 마치며

Whisper는 음성 처리 분야에서 오픈소스의 패러다임을 바꾼 프로젝트입니다. 680,000시간의 데이터로 학습된 denoise 없이도 강건한 인식 능력, 99개 언어 지원, 그리고 단 한 줄 명령어로 시작할 수 있는 편리함은 개발자들의 폭발적인 관심을 받기에 충분했습니다. 인터뷰 전사, 자막 생성, 음성 기반 인터페이스 등 무한한 응용 가능성을 가진 Whisper는 앞으로도 음성 AI 생태계의 든든한 기반이 될 것입니다.
