## 49 Agents IDE란 무엇인가?

AI 에이전트를 여러 개 실행하면 터미널 탭이 넘쳐납니다. 각 머신에 SSH로 접속하고, Alt-Tab으로 에이전트를 확인하고, 어떤 에이전트가 어떤 프로젝트를 하는지 잃어버리곤 합니다. 터미널 14개를 동시에 관리하는 것은 카오스입니다.

**49 Agents IDE**는 이 카오스를 해결합니다. 세계 최초의 2D 에이전트 IDE로, 무한 확장 가능한 캔버스 위에 에이전트, 터미널, 파일 에디터, 이슈 테이블, 노트 등을 자유롭게 배치합니다. 여러 머신의 에이전트도 SSH 없이 하나의 뷰에서 통합 관리합니다.

182개 스타, JavaScript 80.2% + HTML 9.2% + CSS 8.7%, BSL 1.1 라이선스(2030년에 MIT로 전환) 입니다.

| 기존 방식 | 49 Agents IDE |

| 불가 | 인터랙티브 이슈 테이블(Beads) |
| 불가 | 마크다운 노트 |

## 캔버스 기반 워크스페이스 심층 탐구

49 Agents IDE의 핵심은 **무한 확대/축소 가능한 2D 캔버스**입니다.

패인(Pane)들을 마음대로 드래그, 리사이즈, 배치할 수 있습니다. 전체 뷰로 축소하면 모든 에이전트의 상태를 한눈에 확인하고, 확대하면 특정 에이전트의 작업에 집중할 수 있습니다. 레이아웃은 영구 저장되어 재시작 후에도 그대로 유지됩니다.

**패인 종류**: tmux 터미널, Monaco 파일 에디터, Git 그래프, Beads 이슈 테이블, 마크다운 노트, Claude API 사용량 HUD

## 터미널과 멀티 머신

모든 터미널은 **실제 tmux 세션**입니다. ttyd를 통해 브라우저에서 완전한 ANSI 색상, 스크롤백, 쉘 설정이 동작합니다.

**브로드캐스트 입력**: 한번 타이핑으로 여러 터미널에 동시 전송 가능합니다.

**멀티 머신**: Tailscale, LAN, 또는 호스팅 릴레이를 통해 여러 머신의 에이전트를 하나의 캔버스에 연결합니다. SSH 없이. HUD 오버레이로 모든 연결된 머신의 CPU, RAM, Claude API 사용량을 실시간 확인합니다.

## 키보드 퍼스트 네비게이션

49 Agents는 마우스보다 키보드를 우선합니다:

- **탭 코드**: 패인 전환 단축키
- **WASD 이동 모드**: 공간적 탐색
- **1-9 단축키**: 즉시 패인 포커스
- **브로드캐스트 모드**: 멀티 터미널 입력

- **언어**: JavaScript 80.2%, HTML 9.2%, CSS 8.7%, Shell 1.9%
- **라이선스**: BSL 1.1 (2030-02-26에 MIT로 자동 전환)
- **셀프 호스트**: 단일 머신 localhost, Tailscale 클러스터 또는 app.49agents.com

## 설치 및 사용법

```bash
git clone https://github.com/49Agents/49Agents.git
cd 49Agents

# 대화형 설정 (최초 1회)
./49ctl setup

# 클라우드 서버 + 에이전트 시작
./49ctl start
```

`http://localhost:1071`을 열면 됩니다. 계정, 로그인, 토큰이 필요 없습니다.

셀프 호스팅이 귀찮다면 [app.49agents.com](https://app.49agents.com/)에서 호스팅 버전을 사용할 수 있습니다.

## 마치며

49 Agents IDE는 AI 에이전트 관리의 UX 문제를 창의적으로 해결한 프로젝트입니다. 2D 캔버스라는 아이디어는 단순해 보이지만, 여러 에이전트와 머신을 동시에 관리할 때의 인지 부하를 크게 줄입니다. tmux 기반의 실제 터미널, SSH 없는 멀티 머신 연결, Monaco 에디터 통합, Beads 이슈 테이블 등 실용적인 기능들이 잘 결합되어 있습니다. BSL 1.1 라이선스로 개인과 소규모 팀은 무료로 사용 가능하며, 2030년에는 MIT로 완전 오픈됩니다.
