## ST3GG란 무엇인가?

"모든 픽셀에는 이야기가 있습니다. 당신이 볼 수 없을 뿐."

**ST3GG(STE.GG)**는 이미지, 오디오, 문서, 네트워크 패킷, 코드 파일 등 어떤 파일 형식에든 비밀 데이터를 숨길 수 있는 종합 스테가노그래피 툴킷입니다. elder-plinius가 개발한 이 도구는 100가지 이상의 인코딩 기법을 지원하며, 브라우저에서 100% 클라이언트 사이드로 실행됩니다. 서버에 데이터가 전송되지 않고, 모든 처리는 사용자 브라우저에서 이루어집니다.

1,200개 이상의 GitHub 스타를 기록하며, 레드팀/블루팀, CTF 플레이어, 보안 연구자, AI 안전 연구자 등 다양한 사용자층을 보유하고 있습니다.

## 주요 기능 한눈에 보기

ST3GG는 단순한 LSB 스테가노그래피 도구가 아닙니다. 기존 툴 대비 압도적인 기능을 제공합니다:

| 기능 | 기존 툴 | ST3GG |
|------|---------|-------|
| 채널 옵션 | RGB만 | 15개 프리셋 (R, G, B, A, RGB, RGBA 등) |
| 비트 깊이 | 1비트 고정 | 채널당 1-8비트 조정 가능 |
| 인코딩 전략 | 순차 | 4가지 (순차, 인터리브, 분산, 랜덤) |
| 중첩 스테가 | 없음 | 최대 11레이어 (마트료시카 모드) |
| 압축 생존 | 불가 | F5 JPEG/소셜미디어, DCT 저항 |
| 암호화 | 없음/기본 | AES-256-GCM + XOR |
| AI 에이전트 | 없음 | 자동 Reveal/Conceal 에이전트 |

## SPECTER와 마트료시카: 독창적 기법들

ST3GG의 가장 독특한 기능들은 타 도구에서 찾아볼 수 없는 것들입니다.

**SPECTER — 채널 암호화 스테가노그래피**: 데이터 비트가 색상 채널(R, G, B) 사이를 암호화처럼 호핑(hopping)합니다. 예를 들어 `R1-G2-B1-RG2-B1` 패턴은 Red에 1비트, Green에 2비트, Blue에 1비트... 식으로 분산 저장합니다. 패턴 자체가 키가 되는 구조입니다. 수동 패턴 모드와 패스프레이즈 기반 패스워드 모드를 지원합니다.

**Ghost Mode — 최대 은닉**: 3중 난독화로 AES-256-GCM 암호화 → Fisher-Yates 비트 스크램블링 → 50% 노이즈 디코이를 적용합니다. 공격자는 채널 패턴, 해독 패스워드, 암호화 키를 모두 알아야 합니다. 다만 용량이 절반으로 줄어드는 트레이드오프가 있습니다.

**마트료시카 모드**: 이미지 속 이미지 속 이미지... 최대 11레이어까지 재귀적으로 중첩됩니다. 스마트 디코더가 PNG 매직 바이트를 감지하고 모든 레이어를 자동으로 언래핑합니다.

**F5 모드**: JPEG DCT 계수에 직접 임베딩하는 방식으로 소셜미디어 재압축에서도 살아남습니다.

## 레드팀/블루팀 이중 용도 설계

ST3GG는 공격과 방어 양쪽을 위해 설계된 듀얼 유즈 툴킷입니다.

**레드팀 / 공격**: 100가지 이상 기법으로 DLP 방어선을 테스트합니다. DNS 쿼리 이름의 base32 터널링, ICMP 페이로드 인젝션, TCP ISN + 타임스탬프 기반 은닉 채널, HTTP 헤더 밀수(X- 헤더, 쿠키), PNG+ZIP 폴리글랏 파일 등이 포함됩니다.

**블루팀 / 방어 (ALLSIGHT)**: 20개 이상의 탐지 함수(카이제곱 분석, 비트플레인 엔트로피, 히스토그램 분석, STEG 헤더 감지), AI 기반 자동 분석, 유니코드 스테가노그래피 탐지(제로너비 문자, 호모글리프, 변형 선택자), 200개 이상 자동화 테스트로 알려진 기법에서 거짓 음성 0%를 목표로 합니다.

## 기술 스택 및 아키텍처

```
ST3GG/
├── index.html              # 브라우저 UI (100% 클라이언트사이드)
├── steg_core.py            # LSB 인코딩/디코딩 핵심 엔진
├── crypto.py               # AES-256-GCM + XOR 암호화
├── analysis_tools.py       # 20개 이상 탐지 및 분석 함수
├── cli.py                  # 커맨드라인 인터페이스
├── tui.py                  # 텍스추얼 TUI (Textual)
├── webui.py                # 웹 UI (NiceGUI)
├── f5stego-lib.js          # F5 JPEG 스테가노그래피 (브라우저)
└── examples/               # 100개 이상 사전 인코딩 예시 파일
```

- **언어**: HTML 66.9%, Python 30.3%, JavaScript 2.8%
- **라이선스**: AGPL-3.0 (개인/연구/오픈소스 무료, 상용 문의)
- **Python 버전**: 3.9+

## 설치 및 사용법

```bash
# PyPI에서 설치
pip install stegg

# 메시지 숨기기
stegg encode image.png "숨길 메시지" -o stego.png

# 숨겨진 데이터 추출
stegg decode stego.png

# 의심스러운 파일 분석
stegg analyze suspicious.png --full

# TUI 인터페이스
pip install stegg[tui]
stegg-tui

# 웹 UI
pip install stegg[web]
stegg-web

# 브라우저 직접 실행 (서버 불필요)
open index.html
```

SPECTER 패스워드 모드:
```bash
stegg encode image.png "{SPECTER:ENABLED}" -o stego.png
```
