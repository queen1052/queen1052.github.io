## 소개

OpenBB는 OpenBB-finance 팀이 공개한 **오픈소스 금융 분석 플랫폼**입니다. Bloomberg Terminal처럼 강력하지만 구독 비용이 없는 오픈소스 대안을 목표로 만들어졌습니다. 263명의 기여자가 참여한 대형 오픈소스 프로젝트로, 주식, ETF, 암호화폐, 옵션, 포트폴리오 관리를 위한 종합적인 도구를 제공합니다.

Python 기반으로 개인 투자자, 퀀트 개발자, 재무 분석가 모두에게 접근 가능한 도구입니다. CLI(터미널 인터페이스)와 웹 대시보드를 모두 지원하며, 다양한 무료·유료 데이터 소스를 플러그인 형태로 연결할 수 있습니다.

OpenBB의 특징은 확장 가능한 **데이터 공급자 시스템**입니다. Yahoo Finance, Alpha Vantage, Polygon, Federal Reserve(FRED), 암호화폐 거래소 등 수십 개의 데이터 소스를 통일된 API로 접근합니다. "데이터 소스가 바뀌어도 분석 코드는 그대로"라는 설계 철학입니다.

## 주요 기능

- **통합 데이터 공급자 시스템**: Yahoo Finance, Alpha Vantage, Polygon, FRED, Intrinio 등 수십 개의 데이터 소스를 단일 API로 통합합니다. `provider="yfinance"` 처럼 공급자를 파라미터로 지정하면 됩니다.
- **주식 분석 모듈**: 가격 데이터, 재무제표, 애널리스트 추정치, 내부자 거래, 옵션 체인을 분석합니다. 기술적 분석과 기본적 분석을 모두 탑재합니다.
- **포트폴리오 관리**: 다중 자산 포트폴리오의 성과를 추적하고 리스크를 분석합니다. 샤프 비율, 최대 낙폭, 변동성, 상관관계를 자동 계산합니다.
- **암호화폐 분석**: 주요 암호화폐 거래소 데이터를 통합하고 온체인 데이터, 채굴 통계, DeFi 프로토콜 데이터를 분석합니다.
- **거시경제 데이터**: FRED, 세계은행, IMF 등에서 금리, GDP, 물가, 실업률 등 거시경제 지표를 수집하고 시각화합니다.
- **CLI와 Python API 동시 지원**: 대화형 터미널(`openbb`)에서 명령어로 사용하거나, Python 스크립트에서 `obb.equity.price.historical("AAPL")` 처럼 직접 호출합니다.

### 통합 데이터 공급자 — "코드 한 줄로 데이터 소스 전환"

```python
from openbb import obb

# Yahoo Finance로 주가 조회
data_yf = obb.equity.price.historical("AAPL", provider="yfinance")

# Polygon으로 전환 (코드 변경 없이 공급자만 바꿈)
data_poly = obb.equity.price.historical("AAPL", provider="polygon")

# FRED에서 금리 데이터
rates = obb.economy.fred_series("DFF", start_date="2020-01-01")

# 재무제표
income = obb.equity.fundamental.income("MSFT", period="annual")
```

공급자를 바꿔도 반환 형식(OBBject)이 동일하므로 분석 코드 수정이 불필요합니다. 유료 공급자와 무료 공급자를 조합해 비용을 최적화합니다.

### 포트폴리오 분석 — 퀀트 수준의 리스크 분석

```python
# 포트폴리오 설정
portfolio = obb.portfolio.load("my_portfolio.csv")

# 리스크 지표 계산
metrics = obb.portfolio.metrics(portfolio)
print(f"샤프 비율: {metrics.sharpe_ratio:.2f}")
print(f"최대 낙폭: {metrics.max_drawdown:.1%}")
print(f"연간 변동성: {metrics.annual_volatility:.1%}")

# 벤치마크 비교
obb.portfolio.benchmark(portfolio, benchmark="SPY")
```

## 설치 / 사용법

```bash
# pip 설치
pip install openbb

# 전체 플러그인 설치 (주식, 암호화폐, 거시경제 포함)
pip install "openbb[all]"

# CLI 실행
openbb

# OpenBB CLI에서 주식 분석
/stocks/load AAPL --start 2024-01-01
/stocks/fa/income  # 재무제표
/stocks/ta/ema    # 기술적 분석 (EMA)

# Python에서
from openbb import obb
obb.user.preferences.output_type = "dataframe"
df = obb.equity.price.historical("NVDA", provider="yfinance").to_df()
```

## 활용 사례 / 사용 시나리오

1. **개인 투자 분석 자동화**: 관심 종목의 일일 재무 데이터, 뉴스, 내부자 거래를 자동으로 수집하고 분석하는 파이프라인을 구축합니다. Bloomberg Terminal 없이 기관 수준의 분석을 수행합니다.

2. **퀀트 전략 백테스팅**: 기술적 지표와 재무 지표를 결합한 투자 전략을 개발하고 역사적 데이터로 백테스팅합니다. TradingAgents 같은 AI 에이전트와 연동해 AI 트레이딩 전략을 구현합니다.

3. **금융 리포트 자동화**: 특정 섹터나 ETF의 구성 종목 분석, 포트폴리오 월간 리포트, 거시경제 대시보드를 Python으로 자동화합니다. 투자 운용사나 개인 트레이더의 리포팅 부담을 줄입니다.

## 결론

OpenBB는 Bloomberg Terminal의 오픈소스 대안으로서 사용하기 충분한 완성도를 갖추었습니다. 263명의 기여자와 지속적인 업데이트가 생태계의 건강성을 보여줍니다. 통합 데이터 공급자 시스템과 Python/CLI 이중 인터페이스가 개인 투자자부터 기관 퀀트까지 폭넓은 사용자를 수용합니다. 금융 데이터 분석을 자동화하려는 모든 개발자에게 필수 도구입니다.
