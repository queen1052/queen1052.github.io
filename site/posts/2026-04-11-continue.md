## 소개

[Continue](https://github.com/continuedev/continue)는 "소스 제어 AI 체크, CI에서 강제 가능"을 표방하는 혁신적인 개발 도구입니다. 모든 풀 리퀘스트에서 AI 에이전트를 GitHub 상태 체크로 실행하며, 체크 내용은 레포지토리의 마크다운 파일(`.continue/checks/`)로 정의됩니다.

GitHub에서 32.5k 스타를 기록한 Continue는 VS Code/JetBrains 익스텐션으로 잘 알려져 있었지만, 최근 CI 통합 AI 체크 플랫폼으로 진화했습니다. Apache 2.0 라이선스로 오픈소스로 제공되며, 오픈소스 Continue CLI(`cn`)가 모든 AI 체크를 구동합니다.

## 주요 기능

Continue는 AI 코드 리뷰를 팀 레벨의 표준으로 만들어줍니다:

- **PR 상태 체크**: 모든 PR에서 자동으로 AI 에이전트 실행
- **마크다운 체크 정의**: `.continue/checks/`의 `.md` 파일로 체크 규칙 작성
- **CI 강제 실행**: 체크 실패 시 PR 머지 차단 가능
- **자동 수정 제안**: 코드가 기준에 맞지 않으면 AI가 diff를 제안
- **Continue CLI(`cn`)**: 로컬 및 CI 환경에서 체크 실행
- **VS Code/JetBrains 익스텐션**: 개발 중 실시간 AI 지원

## 핵심 기능 심층 분석: 마크다운으로 정의하는 AI 체크

Continue의 가장 독창적인 특징은 AI 체크를 일반 마크다운 파일로 정의한다는 점입니다:

```markdown
---
name: Security Review
description: Review PR for basic security vulnerabilities
---
Review this PR and check that:
- No secrets or API keys are hardcoded
- All new API endpoints have input validation
- Error responses use the standard error format
```

이 파일을 `.continue/checks/security.md`로 저장하면, 모든 PR에서 AI가 보안 검토를 자동으로 수행합니다. 코드가 기준을 충족하면 녹색(Green), 그렇지 않으면 수정 제안과 함께 빨간색(Red) 상태가 됩니다.

더 나아가 팀의 코딩 방식에 맞게 완전히 커스터마이즈됩니다. API 설계 원칙, 테스트 커버리지 기준, 문서화 요건 등 팀의 모든 코딩 기준을 AI 체크로 만들 수 있습니다.

## 핵심 기능 심층 분석: Continue CLI 통합

Continue CLI(`cn`)는 AI 체크를 어디서나 실행하는 핵심 도구입니다:

**설치:**
```bash
# macOS/Linux
curl -fsSL https://raw.githubusercontent.com/continuedev/continue/main/extensions/cli/scripts/install.sh | bash

# npm (Node.js 20+ 필요)
npm i -g @continuedev/cli
```

로컬에서 `cn`을 실행하면 PR 머지 전에 모든 체크를 미리 확인할 수 있습니다. GitHub Actions에 통합하면 모든 PR에서 자동으로 체크가 실행되며, 브랜치 보호 규칙으로 실패한 체크가 있는 PR은 머지를 차단할 수 있습니다.

474명의 컨트리뷰터와 822개의 릴리즈가 말해주는 활발한 개발 속도도 인상적입니다.

## 기술 스택 및 아키텍처

- **언어**: TypeScript (84.4%) + JavaScript (7.4%)
- **모바일**: Kotlin (3.8%) - JetBrains 플러그인
- **백엔드**: Python (2.2%)
- **시스템**: Rust (0.7%)

아키텍처:
```
.continue/checks/     # AI 체크 마크다운 파일
core/                 # 핵심 로직
extensions/           # VS Code, JetBrains, CLI 익스텐션
gui/                  # 웹 UI
packages/             # 공유 패키지
```

## 설치 및 사용법

**시작하기 (AI 에이전트 활용):**
```
Help me write checks for this codebase: https://continue.dev/walkthrough
```
이 프롬프트를 사용 중인 코딩 에이전트에 붙여넣으면 됩니다.

**첫 AI 체크 만들기:**
1. `.continue/checks/` 디렉토리 생성
2. `security.md` 파일 작성 (마크다운 형식)
3. `cn` 명령으로 로컬 실행
4. GitHub Actions에 통합

**VS Code 익스텐션 설치:**
VS Code 마켓플레이스에서 "Continue" 검색 후 설치.

## 마치며

Continue는 AI 코드 리뷰를 팀의 표준 프로세스로 격상시키는 혁신적인 접근법을 제시합니다. 마크다운으로 정의된 AI 체크는 버전 관리되고, 팀이 공유하며, CI에서 강제 실행됩니다. 코드 품질 기준을 문서화하는 동시에 AI가 그 기준을 자동으로 검사합니다. 32.5k 스타와 활발한 기여자 커뮤니티는 Continue가 AI 지원 개발 워크플로우의 미래를 보여주고 있음을 증명합니다.
